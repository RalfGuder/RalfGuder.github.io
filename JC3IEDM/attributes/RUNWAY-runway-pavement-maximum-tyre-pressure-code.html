Attribute
Name: runway-pavement-maximum-tyre-pressure-code
Definition: The specific value that represents the pavement maximum tyre pressure classification, which is part of the standard ICAO (International Civil Aviation Organization) method of reporting pavement strength for pavements with bearing strength greater than 5,700 kilograms (12,500 pounds).
Domain: runway-pavement-maximum-tyre-pressure-code
Other Attribute Properties
This attribute is optional in the entity RUNWAY
