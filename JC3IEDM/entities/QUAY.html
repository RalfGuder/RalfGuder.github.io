Entity
Name: QUAY
Definition: A FACILITY that is a solidly constructed platform, usually parallel to the shoreline of navigable water, alongside which a vessel can be docked or berthed and, on which, the vessel can be accessed and cargo can be loaded or unloaded on one side of the vessel only.
Supertype: FACILITY
Attributes:
Name Opt Definition
quay-id (PK) (FK) MA The facility-id of a specific QUAY (a role name for object-item-id).
quay-container-handling-type-code OP The specific value that represents the class of container handling equipment available at a specific QUAY.
quay-container-maximum-handling-length-dimension OP The one-dimensional linear distance representing the maximum length of a container that can be handled at the QUAY.
quay-container-maximum-handling-weight-quantity OP The numeric value that represents the maximum container weight that can be handled at the QUAY. The unit of measure is kilogram.
quay-crane-offloading-lift-quantity OP The numeric value that represents the maximum weight that can be handled by a crane at the QUAY. The unit of measure is kilogram.
quay-crane-offloading-type-code OP The specific value that represents the class of crane offloading equipment available at a specific QUAY.
quay-day-limit-net-explosive-quantity OP The numeric value that represents the maximum net explosive quantity that can be handled at the QUAY during the day. The unit of measure is kilogram.
quay-draught-dimension OP The one-dimensional linear distance representing the maximum draught of vessel that the specific QUAY can accommodate.
quay-maximum-deadweight-tonnage-quantity OP The numeric value that represents the maximum deadweight tonnage that can be accommodated for a vessel at the specific QUAY. The unit of measure is metric ton.
quay-night-limit-net-explosive-quantity OP The numeric value that represents the maximum net explosive quantity that can be handled at the QUAY during the night. The unit of measure is kilogram.
quay-rail-capacity-count OP The integer value representing the maximum number of goods/freight cars that the QUAY can accommodate at the same time.
quay-rail-served-indicator-code OP The specific value that represents whether the QUAY has railway facilities.
quay-storage-code OP The specific value that represents the class of storage facilities available at a specific QUAY.
quay-vessel-maximum-beam-dimension OP The one-dimensional linear distance representing the maximum beam or width of vessel that the specific QUAY can accommodate.
