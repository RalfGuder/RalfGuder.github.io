Entity
Name: ACTION-MARITIME-EMPLOYMENT
Definition: The procedure that guides the use of an ACTION-RESOURCE in a maritime environment.
Supertype: ACTION-RESOURCE-EMPLOYMENT
Attributes:
Name Opt Definition
action-id (PK) (FK) MA The unique value, or set of characters, assigned to represent a specific ACTION and to distinguish it from all other ACTIONs.
action-resource-employment-index (PK) (FK) MA The unique value, or set of characters, assigned to represent a specific ACTION-RESOURCE-EMPLOYMENT for a specific ACTION-RESOURCE and to distinguish it from all other ACTION-RESOURCE-EMPLOYMENTs for that ACTION-RESOURCE.
action-resource-index (PK) (FK) MA The unique value, or set of characters, assigned to represent a specific ACTION-RESOURCE for a specific ACTION and to distinguish it from all other ACTION-RESOURCEs for that ACTION.
action-maritime-employment-coordinated-air-sea-procedure-code OP The specific value that represents the coordinated air sea procedure (CASP) category for a friendly force, indicating the nature of the force relationship to the air commander.
action-maritime-employment-group-vessel-transit-longitudinal-spacing-dimension OP The one dimensional linear measurement representing the distance between groups of vessels in order to traverse a specific maritime minefield. The distance is measured in nautical miles.
action-maritime-employment-lead-vessel-name-text OP An unformatted character string assigned to name a specific lead vessel in a convoy moving through a maritime minefield.
action-maritime-employment-number-runs-required-count OP An integer value that represents the number of MCM runs required through a minefield to achieve the required clearance level.
action-maritime-employment-swept-lane-actuation-width-dimension OP The one-dimensional linear measurement representing the swept lane width within which mines will be actuated.
action-maritime-employment-vessel-transit-instruction-code OP The specific value providing lead through instructions for a convoy or vessel to pass through a maritime minefield.
action-maritime-employment-vessel-transit-longitudinal-spacing-dimension OP The one dimensional linear measurement representing the distance between vessels in order to traverse a specific maritime minefield. The distance is measured in metres.
action-maritime-employment-vessel-transit-recommended-speed-rate OP The numeric value that represents the maximum distance per unit time that is recommended for a vessel to traverse a specific maritime minefield. The speed is measured in knots. The specified value must be greater than or equal to 0 (zero).
