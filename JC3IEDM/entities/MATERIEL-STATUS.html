Entity
Name: MATERIEL-STATUS
Definition: An OBJECT-ITEM-STATUS that is a record of condition of a specific MATERIEL.
Supertype: OBJECT-ITEM-STATUS
Subtypes: MINE-STATUS, UXO-STATUS
Attributes:
Name Opt Definition
materiel-status-id (PK) (FK) MA The materiel-id of the MATERIEL that is the subject of a specific MATERIEL-STATUS (a role name for object-item-id).
object-item-status-index (PK) (FK) MA The unique value, or set of characters, assigned to represent a specific OBJECT-ITEM-STATUS for a specific OBJECT-ITEM and to distinguish it from all other OBJECT-ITEM-STATUSs for that OBJECT-ITEM.
materiel-status-body-colour-code OP The specific value that represents the current colour scheme of a specific MATERIEL.
materiel-status-buoy-malfunction-code OP The specific value that represents the type of malfunction of a buoy.
materiel-status-category-code OP The specific value that represents the class of MATERIEL-STATUS. It serves as a discriminator that partitions MATERIEL-STATUS into subtypes.
materiel-status-demolition-status-code OP The specific value that represents the status of a specific MATERIEL destined for demolition.
materiel-status-imo-compliant-indicator-code OP The specific value that indicates whether a vessel complies with International Maritime Organisation standards.
materiel-status-marking-code OP The specific value that represents the type of marking of a specific MATERIEL.
materiel-status-marking-colour-code OP The specific value that represents the colour of the markings of a specific MATERIEL.
materiel-status-operational-status-code OP The specific value that represents the operational status of a specific MATERIEL.
materiel-status-operational-status-mode-code OP The specific value that represents the firepower or mobility or communications degradation of a specific MATERIEL.
materiel-status-operational-status-qualifier-code OP The specific value that represents the qualification of the operational status of a specific MATERIEL.
materiel-status-reserve-indicator-code OP The specific value that represents whether a specific MATERIEL has been placed in reserve.
materiel-status-safety-status-code OP The specific value that represents the arming state of a specific MATERIEL that is a weapon or ammunition.
materiel-status-usage-status-code OP The specific value that represents the usage of a specific MATERIEL.
