Entity
Name: CHEMICAL-BIOLOGICAL-EVENT
Definition: A CBRN-EVENT involving chemical and/or biological materiel.
Supertype: CBRN-EVENT
Attributes:
Name Opt Definition
chemical-biological-event-id (PK) (FK) MA The cbrn-event-id of a specific CHEMICAL-BIOLOGICAL-EVENT (a role name for action-id).
chemical-biological-event-atmospheric-particle-concentration-quantity OP The numeric value that represents the level of chemical or biological atmospheric contamination The unit of measure is Agent Containing Particles Per Litre (ACPL).
chemical-biological-event-category-code OP The specific value that represents the class of CHEMICAL-BIOLOGICAL-EVENT.
chemical-biological-event-mass-fraction-concentration-quantity OP The numeric value that represents the number of 'parts' by weight of a biological or chemical substance per million parts of water. The unit of measure is in Parts per Million (PPM).
chemical-biological-event-release-category-code OP The specific value that represents the class of release in a CHEMICAL-BIOLOGICAL-EVENT.
chemical-biological-event-release-height-dimension OP The one-dimensional linear distance representing the height above ground level at which the chemical or biological agent is released. Release height is frequently referred to as burst height.
chemical-biological-event-spill-size-code OP The specific value that represents the mass or the volume of a materiel spilled in a CHEMICAL-BIOLOGICAL-EVENT that is a release other than attack (ROTA).
chemical-biological-event-surface-deposition-quantity OP The numeric value that represents the level of chemical or biological contamination per unit area on a surface. The unit of measure is milligrams per square metre (MGM2).
chemical-biological-event-volume-concentration-quantity OP The numeric value that represents the level of chemical or biological contamination per unit volume in air or water. The unit of measure is milligrams per cubic metre (MGM3).
