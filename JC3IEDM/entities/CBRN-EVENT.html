Entity
Name: CBRN-EVENT
Definition: An ACTION-EVENT that involves chemical, biological, radiological or nuclear materiel individually or in combination.
Supertype: ACTION-EVENT
Subtypes: CHEMICAL-BIOLOGICAL-EVENT, RADIOACTIVE-EVENT
Attributes:
Name Opt Definition
cbrn-event-id (PK) (FK) MA The action-event-id of a specific CBRN-EVENT (a role name for action-id).
cbrn-event-alarm-result-indicator-code OP The specific value that denotes whether a detector has indicated the presence of a CBRN-EVENT.
cbrn-event-category-code OP The specific value that represents the class of CBRN-EVENT. It serves as a discriminator that partitions CBRN-EVENT into subtypes.
cbrn-event-confirmation-test-indicator-code OP The specific value that denotes whether a test confirms the presence of a CBRN-EVENT.
cbrn-event-materiel-container-total-count OP The integer value representing the number of [materiel] agent containers involved in a specific CBRN-EVENT and characterised in ATP-45.
cbrn-event-materiel-container-type-code OP The specific value that represents the type of container that stores the materiel (agent) involved in a specific CBRN-EVENT and characterised in ATP-45.
cbrn-event-subcategory-code OP The specific value that represents the detailed class or nature of activity prescribed by CBRN-EVENT.
