Entity
Name: MINEFIELD-MARITIME

Supertype: MINEFIELD

Attributes:
Name Opt Definition
minefield-maritime-id (PK) (FK) MA The minefield-id of a specific MINEFIELD-MARITIME (a role name for object-item-id).
minefield-maritime-bottom-natural-camouflage-code OP The specific value that represents the description of the ground of a lake, river, or ocean, or other body of water with respect to the ability to hide mines on the bottom at a specific MINEFIELD-MARITIME.
minefield-maritime-depth-placement-code OP The specific value that indicates the intended depth placement of maritime mines.
minefield-maritime-detection-probability-ratio OP The numeric quotient value that represents the probability that a mine will be located during MCM operations. The value must be in the range from 0 to 1.
minefield-maritime-expected-vessel-transit-count OP The integer value representing the expected number of transiting vessels.
minefield-maritime-function-code OP The specific value that represents the intended function of a specific MINEFIELD-MARITIME.
minefield-maritime-life-duration OP The numeric value that represents a quantity of time in milliseconds for the estimated life of the MINEFIELD-MARITIME.
minefield-maritime-mine-detailed-text OP The character string assigned to represent a description of a specific mine.
minefield-maritime-mines-laid-count OP The integer value representing the number of maritime mines laid in a specific MINEFIELD-MARITIME.
minefield-maritime-mmoe-initial-threat-probability-ratio OP The numeric quotient value that represents the planned or estimated likelihood that the first target ship to enter the minefield will be a casualty. MMOE stands for Minefield Measurement Of Effectiveness. The value must be in the range from 0 to 1.
minefield-maritime-subfunction-code OP The specific value that represents the intended purpose of the MINEFIELD-MARITIME.
