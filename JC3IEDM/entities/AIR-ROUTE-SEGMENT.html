Entity
Name: AIR-ROUTE-SEGMENT
Definition: A portion of a route to be flown usually without an intermediate stop, as defined by two consecutive significant points.
Supertype: ROUTE-SEGMENT
Attributes:
Name Opt Definition
air-route-segment-id (PK) (FK) MA The route-segment-id of a specific AIR-ROUTE-SEGMENT (a role name for object-item-id).
air-route-segment-civil-military-code OP The specific value that represents the civil/military status of the AIR-ROUTE-SEGMENT.
air-route-segment-description-text OP The character string assigned to represent the description of specific AIR-ROUTE-SEGMENT.
air-route-segment-international-route-code OP The specific value that represents the domestic/international status of the AIR-ROUTE-SEGMENT.
air-route-segment-maintained-speed-rate OP The numeric value that denotes the speed of movement to be maintained between route points. The unit of measure is kilometres per hour. The specified value must be greater than or equal to 0 (zero).
air-route-segment-required-navigation-performance-code OP The specific value that represents the required navigation performance when flying routes for which external route navigation aids are not provided.
