Entity
Name: MINEFIELD-MARITIME-STATUS
Definition: A FACILITY-STATUS that is a record of condition of a specific MINEFIELD-MARITIME.
Supertype: FACILITY-STATUS
Attributes:
Name Opt Definition
minefield-maritime-status-id (PK) (FK) MA The facility-id of the MINEFIELD-MARITIME that is the subject of a specific MINEFIELD-MARITIME-STATUS (a role name for object-item-id).
object-item-status-index (PK) (FK) MA The unique value, or set of characters, assigned to represent a specific OBJECT-ITEM-STATUS for a specific OBJECT-ITEM and to distinguish it from all other OBJECT-ITEM-STATUSs for that OBJECT-ITEM.
minefield-maritime-status-code OP The specific value that represents the known status of a MINEFIELD-MARITIME.
minefield-maritime-status-colour-code OP The specific value that represents the known status of mined channels of a MINEFIELD-MARITIME.
minefield-maritime-status-expected-level-mcm-code OP The specific values of expected mine countermeasures (MCM) to be employed against a MINEFIELD-MARITIME.
minefield-maritime-status-mine-detection-code OP The specific value that represents the status of the means of detection of a mine in a MINEFIELD-MARITIME.
minefield-maritime-status-mine-zone-risk-code OP The specific value that represents the known threat (risk) of a MINEFIELD-MARITIME.
minefield-maritime-status-mines-count OP The integer value representing the number of mines within a specific maritime minefield at the time of the report.
minefield-maritime-status-mines-detected-count OP The integer value representing the number of mines detected in a specific MINEFIELD-MARITIME at the time of the report.
minefield-maritime-status-mines-detected-quantity OP The numeric value representing the number of mines detected in a specific MINEFIELD-MARITIME.
minefield-maritime-status-seeding-code OP The specific value that indicates the seeding details for a MINEFIELD-MARITIME.
minefield-maritime-status-swept-depth-quantity OP The numeric value representing the depth below the surface to which minesweeping has taken place. The unit of measure is metres.
minefield-maritime-status-threat-ratio OP The numeric quotient value that represents the current percentage threat to the enemy of a specific MINEFIELD-MARITIME. The value must be in the range from 0 to 1.
