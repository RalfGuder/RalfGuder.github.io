Entity
Name: MINEFIELD
Definition: A MILITARY-OBSTACLE that is an area or volume containing mines.
Supertype: MILITARY-OBSTACLE
Subtypes: MINEFIELD-LAND, MINEFIELD-MARITIME
Attributes:
Name Opt Definition
minefield-id (PK) (FK) MA The military-obstacle-id of a specific MINEFIELD (a role name for object-item-id).
minefield-category-code OP The specific value that represents the class of MINEFIELD. It serves as a discriminator that partitions MINEFIELD into subtypes.
minefield-destruction-datetime OP The character string representing a point in time that designates the destruction of a specific MINEFIELD.
minefield-identification-text OP The character string assigned to represent the designation of a minefield.
minefield-mine-spacing-dimension OP The one-dimensional linear distance representing the distance between the mines emplaced in a specific MINEFIELD.
