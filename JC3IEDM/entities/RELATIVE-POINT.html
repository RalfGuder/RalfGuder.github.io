Entity
Name: RELATIVE-POINT
Definition: A POINT whose position is specified with respect to a specific RELATIVE-COORDINATE-SYSTEM.
Supertype: POINT
Cardinality relationships:
Parent Entity Verb Phrase Child Entity Relationship Type Logical Foreign Keys Cardinality
RELATIVE-COORDINATE-SYSTEM used-for-expression-of / expressed-with-reference-to RELATIVE-POINT Non-identifying relative-coordinate-system-id One-to-Zero-One-or-More
Attributes:
Name Opt Definition
relative-point-id (PK) (FK) MA The point-id of a specific RELATIVE-POINT (a role name for location-id).
relative-coordinate-system-id (FK) OP The unique value, or set of characters, assigned to represent a specific RELATIVE-COORDINATE-SYSTEM and to distinguish it from all other RELATIVE-COORDINATE-SYSTEMs.
relative-point-x-coordinate-dimension OP The one-dimensional linear distance representing the displacement of the specific RELATIVE-POINT along the x-axis with respect to a specific RELATIVE-COORDINATE-SYSTEM.
relative-point-x-precision-code OP The specific value that represents the maximum resolution used for the expression of a value of an x-coordinate.
relative-point-y-coordinate-dimension OP The one-dimensional linear distance representing the displacement of the specific RELATIVE-POINT along the y-axis with respect to a specific RELATIVE-COORDINATE-SYSTEM.
relative-point-y-precision-code OP The specific value that represents the maximum resolution used for the expression of a value of a y-coordinate.
relative-point-z-coordinate-dimension OP The one-dimensional linear distance representing the displacement of the specific RELATIVE-POINT along the z-axis with respect to a specific RELATIVE-COORDINATE-SYSTEM.
relative-point-z-precision-code OP The specific value that represents the maximum resolution used for the expression of a value of a z-coordinate.
