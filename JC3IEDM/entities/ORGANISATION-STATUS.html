Entity
Name: ORGANISATION-STATUS
Definition: An OBJECT-ITEM-STATUS that is a record of condition of a specific ORGANISATION.
Supertype: OBJECT-ITEM-STATUS
Attributes:
Name Opt Definition
object-item-status-index (PK) (FK) MA The unique value, or set of characters, assigned to represent a specific OBJECT-ITEM-STATUS for a specific OBJECT-ITEM and to distinguish it from all other OBJECT-ITEM-STATUSs for that OBJECT-ITEM.
organisation-status-id (PK) (FK) MA The organisation-id of the ORGANISATION that is the subject of a specific ORGANISATION-STATUS (a role name for object-item-id).
organisation-status-availability-code OP The specific value that gives the availability status of an ORGANISATION without regard to readiness.
organisation-status-cbrn-dress-state-code OP The specific value that represents the Mission Oriented Protective Posture (MOPP) status defining the NBC (CBRN) dress code of a specific ORGANISATION.
organisation-status-command-and-control-role-code OP The specific value that represents the role played by a command and control ORGANISATION.
organisation-status-commitment-status-code OP The specific value that gives the commitment status of an ORGANISATION.
organisation-status-fire-mode-code OP The specific value that represents the status of weapons employment constraint for a specific ORGANISATION.
organisation-status-operational-status-code OP The specific value that represents the operational status of a specific ORGANISATION.
organisation-status-operational-status-qualifier-code OP The specific value that represents the qualification of the operational status of a specific ORGANISATION.
organisation-status-radiation-dose-quantity OP The numeric value that represents the average radiation dose to which the members of an organisation have been exposed. The unit of measure is centiGray (cGy).
organisation-status-readiness-code OP The specific value that gives the readiness level of an ORGANISATION.
organisation-status-readiness-duration OP The numeric value that represents a quantity of time in milliseconds reflecting the maximum interval in which an ORGANISATION is to respond to an immediate order.
organisation-status-reinforcement-code OP The specific value that represents whether a specific ORGANISATION has additional or detached strength.
organisation-status-reserve-indicator-code OP The specific value that represents whether a specific ORGANISATION has been placed in reserve.
organisation-status-training-code OP The specific value that represents the assessed training status of a specific ORGANISATION.
organisation-status-usage-status-code OP The specific value that represents the usage of a specific ORGANISATION.
