Entity
Name: CONSUMABLE-MATERIEL-TYPE
Definition: A MATERIEL-TYPE that is an expendable class of supply.
Supertype: MATERIEL-TYPE
Subtypes: AMMUNITION-TYPE, BIOLOGICAL-MATERIEL-TYPE, CHEMICAL-MATERIEL-TYPE, RADIOACTIVE-MATERIEL-TYPE
Attributes:
Name Opt Definition
consumable-materiel-type-id (PK) (FK) MA The materiel-type-id of a specific CONSUMABLE-MATERIEL-TYPE (a role name for object-type-id).
consumable-materiel-type-category-code OP The specific value that represents the class of CONSUMABLE-MATERIEL-TYPE. It serves as a discriminator that partitions CONSUMABLE-MATERIEL-TYPE into subtypes.
consumable-materiel-type-hazard-code OP The specific value that represents a CONSUMABLE-MATERIEL-TYPE that requires special handling because of environmental or safety reasons.
consumable-materiel-type-issuing-count OP The integer value representing the aggregated units in which a specific CONSUMABLE-MATERIEL-TYPE is made available.
consumable-materiel-type-issuing-element-code OP The specific value that represents a standard unit in which a specific CONSUMABLE-MATERIEL-TYPE is made available.
consumable-materiel-type-issuing-unit-of-measure-code OP The specific value that represents the unit of measure of which a standard quantity (unit) of a specific CONSUMABLE-MATERIEL-TYPE is made available.
consumable-materiel-type-issuing-weight-quantity OP The numeric value that represents the gravitational force exerted on an amount of a standard unit of issue for a specific CONSUMABLE-MATERIEL-TYPE when it is prepared for delivery. The unit of measure is kilogram.
consumable-materiel-type-perishability-indicator-code OP The specific value that represents whether a particular CONSUMABLE-MATERIEL-TYPE is liable to decay or spoil.
consumable-materiel-type-subcategory-code OP The specific value that represents the detailed class of a specific CONSUMABLE-MATERIEL-TYPE.
consumable-materiel-type-united-nations-number-code OP The specific value that represents the United Nations (UN) Numbers that are four-digit numbers used world-wide in international commerce and transportation to identify hazardous chemicals or classes of hazardous materials.
