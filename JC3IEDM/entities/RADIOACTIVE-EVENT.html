Entity
Name: RADIOACTIVE-EVENT
Definition: A CBRN-EVENT involving radioactive materiel(s).
Supertype: CBRN-EVENT
Subtypes: NUCLEAR-EVENT, RADIOLOGICAL-EVENT
Attributes:
Name Opt Definition
radioactive-event-id (PK) (FK) MA The cbrn-event-id of a specific RADIOACTIVE-EVENT (a role name for action-id).
radioactive-event-category-code OP The specific value that represents the class of RADIOACTIVE-EVENT. It serves as a discriminator that partitions RADIOACTIVE-EVENT into subtypes.
radioactive-event-dose-quantity OP The numeric value that represents the total radiation dose accumulated over the duration of the RADIOACTIVE-EVENT. The unit of measure is centiGray (cGy).
radioactive-event-dose-rate OP The numeric value that denotes the radiation dose rate. The unit of measure is centiGray per Hour (cGy/h). The specified value must be greater than or equal to 0 (zero).
radioactive-event-dose-rate-trend-code OP The specific value that represents the rate of change of the ionising radiation emitted by a radioactive materiel.
radioactive-event-relative-decay-rate-code OP The specific value that represents the rate of decay of fallout relative to the assumed normal value of 1.2 in the Kaufmann equation.
radioactive-event-surface-deposition-quantity OP The numeric value that represents the level of radioactive contamination per unit area on a surface. The unit of measure is Becquerels per square metre (BQM2).
radioactive-event-volume-concentration-quantity OP The numeric value that represents the level of radioactive contamination per unit volume in air or water. The unit of measure is Becquerels per cubic metre (BQM3).
