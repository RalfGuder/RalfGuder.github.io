Entity
Name: GEOGRAPHIC-FEATURE
Definition: A FEATURE describing terrain characteristics to which military significance is attached.
Supertype: FEATURE
Attributes:
Name Opt Definition
geographic-feature-id (PK) (FK) MA The feature-id of a specific GEOGRAPHIC-FEATURE (a role name for object-item-id).
geographic-feature-bottom-hardness-code OP A specific value that represents the subjective indication obtained by a diver of the hardness of the liquid/solid surface interface and is the result of a single arm thrust.
geographic-feature-bottom-penetration-quantity OP The numeric value that represents the depth to which a diver is able to thrust his fist or fingers into the surface of the solid/liquid interface. The unit of measure is metres.
geographic-feature-solid-surface-composition-code OP The specific value that represents the composition of the surface of the GEOGRAPHIC-FEATURE.
geographic-feature-surface-category-code OP The specific value that represents the type of surface of the GEOGRAPHIC-FEATURE.
geographic-feature-terrain-code OP The specific value that represents a tract of land.
geographic-feature-vegetation-category-code OP The specific value that represents the general vegetation class on a specific GEOGRAPHIC-FEATURE.
geographic-feature-vegetation-subcategory-code OP The specific value that represents the detailed vegetation class on a specific GEOGRAPHIC-FEATURE.
