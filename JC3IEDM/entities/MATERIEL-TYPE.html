Entity
Name: MATERIEL-TYPE
Definition: An OBJECT-TYPE that represents equipment, apparatus or supplies of military interest without distinction to its application for administrative or combat purposes.
Supertype: OBJECT-TYPE
Subtypes: CONSUMABLE-MATERIEL-TYPE, EQUIPMENT-TYPE
Cardinality relationships:
Parent Entity Verb Phrase Child Entity Relationship Type Logical Foreign Keys Cardinality
MATERIEL-TYPE is-assigned-reporting-code-in / ORGANISATION-MATERIEL-TYPE-ASSOCIATION Identifying materiel-type-id One-to-Zero-One-or-More
Attributes:
Name Opt Definition
materiel-type-id (PK) (FK) MA The object-type-id of a specific MATERIEL-TYPE (a role name for object-type-id).
materiel-type-category-code OP The specific value that represents the class of MATERIEL-TYPE. It serves as a discriminator that partitions MATERIEL-TYPE into subtypes.
materiel-type-issuing-height-dimension OP The one-dimensional linear distance representing the maximum distance measured perpendicular to the base plane of the specific MATERIEL-TYPE when it is prepared for shipment or storage.
materiel-type-issuing-length-dimension OP The one-dimensional linear distance representing the maximum distance measured from end to end and parallel to the central axis of specific MATERIEL-TYPE when it is prepared for shipment or storage.
materiel-type-issuing-width-dimension OP The one-dimensional linear distance representing the maximum distance measured from side to side and perpendicular to the central axis of a specific MATERIEL-TYPE when it is prepared for shipment or storage.
materiel-type-reportable-item-text OP The character string assigned to represent a specific MATERIEL-TYPE, selected from the Reportable Item Code list issued by NATO.
materiel-type-stock-number-text OP The character string assigned to represent a specific MATERIEL-TYPE.
materiel-type-supply-class-code OP The specific value that represents the NATO supply class of MATERIEL-TYPE.
