Entity
Name: CLOUD-COVER
Definition: A METEOROLOGIC-FEATURE that specifies the characteristics of clouds above Earth's surface.
Supertype: METEOROLOGIC-FEATURE
Attributes:
Name Opt Definition
cloud-cover-id (PK) (FK) MA The meteorologic-feature-id of a specific CLOUD-COVER (a role name for object-item-id).
cloud-cover-average-coverage-code OP The specific value that represents the average density of a specific CLOUD-COVER as fractional coverage.
cloud-cover-base-dimension OP The one-dimensional linear distance representing the elevation of the lowest cloud base for a specific CLOUD-COVER.
cloud-cover-category-code OP The specific value that represents the prevailing class of a specific CLOUD-COVER.
cloud-cover-light-refraction-ratio OP The numeric quotient value that represents the velocity of light in a specific CLOUD-COVER as a fraction of the velocity of light in a vacuum, based on cloud height. This represents the inverse of the index of refraction. The value must be in the range from 0 to 1.
cloud-cover-top-dimension OP The one-dimensional linear distance representing the elevation of the highest cloud ceiling for a specific CLOUD-COVER.
