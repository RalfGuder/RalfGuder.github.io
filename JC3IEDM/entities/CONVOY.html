Entity
Name: CONVOY
Definition: An ORGANISATION that is a group of vehicles or vessels organised for the purpose of control and orderly movement with or without escort protection.
Supertype: ORGANISATION
Attributes:
Name Opt Definition
convoy-id (PK) (FK) MA The organisation-id of a specific CONVOY (a role name for object-item-id).
convoy-day-speed-rate OP The numeric value that denotes the maximum distance per unit time that a specific CONVOY is to maintain during daylight operations. The unit of measure is kilometres per hour. The specified value must be greater than or equal to 0 (zero).
convoy-day-vehicle-gap-dimension OP The one-dimensional linear distance representing the distance between vehicles in a particular CONVOY during daylight operations.
convoy-halt-duration OP The numeric value that represents a quantity of time in milliseconds representing the aggregated units of time that a specific CONVOY may remain stationary during operations.
convoy-night-speed-rate OP The numeric value that denotes the maximum distance per unit time that a specific CONVOY is to maintain during operations in darkness. The unit of measure is kilometres per hour. The specified value must be greater than or equal to 0 (zero).
convoy-night-vehicle-gap-dimension OP The one-dimensional linear distance representing the distance between vehicles in a particular CONVOY during operations in darkness.
convoy-packet-gap-dimension OP The one-dimensional linear distance representing the distance between packets in a particular CONVOY.
convoy-packet-size-count OP The integer value that represents the number of vehicles per packet in a particular CONVOY.
