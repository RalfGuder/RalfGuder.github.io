Entity
Name: CARTESIAN-POINT
Definition: An ABSOLUTE-POINT that has its position specified in a three-dimensional Earth-centred Cartesian system.
Supertype: ABSOLUTE-POINT
Attributes:
Name Opt Definition
cartesian-point-id (PK) (FK) MA The absolute-point-id of a specific CARTESIAN-POINT (a role name for location-id).
cartesian-point-x-coordinate-dimension OP The one-dimensional linear distance representing the X-component of a coordinate which expresses the position of a point in a three-dimensional Cartesian coordinate system that is fixed to the earth, where the X-axis lies in the planes of the Equator and the Greenwich meridian.
cartesian-point-x-precision-code OP The specific value that represents the resolution used for the expression of a value of a Cartesian x-coordinate.
cartesian-point-y-coordinate-dimension OP The one-dimensional linear distance representing the Y-component of a coordinate which expresses the position of a point in a three-dimensional Cartesian coordinate system that is fixed to the earth where the Y-axis is perpendicular to both the X- and Z-axes completing the right-handed coordinate system.
cartesian-point-y-precision-code OP The specific value that represents the resolution used for the expression of a value of a Cartesian y-coordinate.
cartesian-point-z-coordinate-dimension OP The one-dimensional linear distance representing the Z-component of a coordinate which expresses the position of a point in a three-dimensional Cartesian coordinate system that is fixed to the earth, where the Z-axis coincides with the mean rotation axis of the Earth.
cartesian-point-z-precision-code OP The specific value that represents the resolution used for the expression of a value of a Cartesian z-coordinate.
