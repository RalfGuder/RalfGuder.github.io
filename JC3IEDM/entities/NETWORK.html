Entity
Name: NETWORK
Definition: A FACILITY that provides bearer services for communication and information services and is composed of one or more links and nodes.
Supertype: FACILITY
Cardinality relationships:
Parent Entity Verb Phrase Child Entity Relationship Type Logical Foreign Keys Cardinality

NETWORK provides / is-provided-by NETWORK-SERVICE Identifying network-id One-to-Zero-One-or-More
NETWORK has / is-specified-for NETWORK-CAPACITY Identifying network-id One-to-Zero-One-or-More
Attributes:
Name Opt Definition
network-id (PK) (FK) MA The facility-id of a specific NETWORK (a role name for object-item-id).
network-architecture-code OP The specific value that represents the architecture of a specific NETWORK.
network-category-code OP The specific value that represents the class of NETWORK.
network-channel-count OP The integer value representing the number of channels that a specific NETWORK can provide.
network-maximum-capacity-rate OP The numeric value that represents the maximum data rate that a specific NETWORK can process. The unit of measure is kilobits per second (KBS). The specified value must be greater than or equal to 0 (zero).
network-means-code OP The specific value that represents the physical linkage between the nodes of a specific NETWORK.
network-minimum-capacity-rate OP The numeric value that represents the minimum data rate that a specific NETWORK is required to process. The unit of measure is kilobits per second (KBS). The specified value must be greater than or equal to 0 (zero).
network-set-number-count OP The integer value representing the frequency hopping set number parameter for this specific NETWORK.
network-subcategory-code OP The specific value that represents the specific class of NETWORK.
