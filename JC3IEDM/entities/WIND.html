Entity
Name: WIND
Definition: A METEOROLOGIC-FEATURE that specifies the velocity and directional characteristics of atmospheric movement.
Supertype: METEOROLOGIC-FEATURE
Attributes:
Name Opt Definition
wind-id (PK) (FK) MA The meteorologic-feature-id of a specific WIND (a role name for object-item-id).
wind-air-stability-category-code OP The specific value used to indicate the class of air stability.
wind-altitude-layer-code OP The specific value used to indicate the class of the altitude for a specific set of reported wind data.
wind-category-code OP The specific value that represents the class of WIND.
wind-direction-angle OP The rotational measurement clockwise between the line of true North and the direction from which a specific WIND originates.
wind-effective-downwind-direction-angle OP The rotational measurement of the mean downwind direction at ground level in the hazard area towards which the cloud travels.
wind-nuclear-yield-qualifier-code OP The specific value used to identify predicted wind values associated with nuclear fallout prediction for a specific Nuclear yield group.
wind-speed-rate OP The numeric value that denotes the distance per unit time of a specific WIND. The unit of measure is kilometres per hour. The specified value must be greater than or equal to 0 (zero).
