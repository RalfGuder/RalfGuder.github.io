Entity
Name: PHYSICAL-ADDRESS
Definition: An ADDRESS that represents a physical location that is reachable by use of transportation, to include the use of postal services.
Supertype: ADDRESS
Attributes:
Name Opt Definition
address-id (PK) (FK) MA The unique value, or set of characters, assigned to represent a specific ADDRESS and to distinguish it from all other ADDRESSs.
physical-address-category-code OP The specific value that represents the class of the PHYSICAL-ADDRESS.
physical-address-city-text OP The character string assigned to represent the city name of the PHYSICAL-ADDRESS.
physical-address-district-text OP The character string assigned to represent the district of the PHYSICAL-ADDRESS.
physical-address-geographic-text OP The character string assigned to represent the geographic region of the PHYSICAL-ADDRESS.
physical-address-postal-box-text OP The character string assigned to represent a specific post office box of the PHYSICAL-ADDRESS.
physical-address-postal-code-text OP The character string assigned to represent the postal code of the PHYSICAL-ADDRESS.
physical-address-postbox-identifier-text OP The character string assigned to represent a specific letter box number of the PHYSICAL-ADDRESS.
physical-address-province-text OP The character string assigned to represent the province of the PHYSICAL-ADDRESS.
physical-address-residence-text OP The character string assigned to represent the residence name of the PHYSICAL-ADDRESS.
physical-address-street-additional-text OP The character string assigned to represent specific additional information such as the house number, the apartment number, rural route number, building number, room number, office or equivalent number to complete the physical address for a specific PHYSICAL-ADDRESS.
physical-address-street-text OP The character string assigned to represent the street name for a specific PHYSICAL-ADDRESS.
