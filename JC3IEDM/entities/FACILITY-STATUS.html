Entity
Name: FACILITY-STATUS
Definition: An OBJECT-ITEM-STATUS that is a record of condition of a specific FACILITY.
Supertype: OBJECT-ITEM-STATUS
Subtypes: AIRFIELD-STATUS, MEDICAL-FACILITY-STATUS, MINEFIELD-MARITIME-STATUS
Attributes:
Name Opt Definition
facility-status-id (PK) (FK) MA The facility-id of the FACILITY that is the subject of a specific FACILITY-STATUS (a role name for object-item-id).
object-item-status-index (PK) (FK) MA The unique value, or set of characters, assigned to represent a specific OBJECT-ITEM-STATUS for a specific OBJECT-ITEM and to distinguish it from all other OBJECT-ITEM-STATUSs for that OBJECT-ITEM.
facility-status-category-code OP The specific value that represents the class of FACILITY-STATUS. It serves as a discriminator that partitions FACILITY-STATUS into subtypes.
facility-status-demolition-status-code OP The specific value that represents the status of a specific FACILITY destined for demolition.
facility-status-enemy-activity-condition-code OP The specific value that represents the status of enemy activity around or at the FACILITY.
facility-status-mine-presence-code OP The specific value that indicates whether a specific FACILITY contains mines.
facility-status-occupation-program-indicator-code OP The specific value that indicates whether an occupation programme is present at the facility.
facility-status-operational-status-code OP The specific value that represents the operational status of a specific FACILITY.
facility-status-operational-status-qualifier-code OP The specific value that represents the qualification of the operational status of a specific FACILITY.
facility-status-reserve-indicator-code OP The specific value that represents whether a specific FACILITY has been placed in reserve.
facility-status-security-status-code OP The specific value that represents the security status of a specific FACILITY.
facility-status-usage-status-code OP The specific value that represents the usage of a specific FACILITY.
