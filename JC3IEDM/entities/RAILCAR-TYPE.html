Name: RAILCAR-TYPE
Definition: An EQUIPMENT-TYPE that is designed to operate on rail tracks.
Supertype: EQUIPMENT-TYPE
Attributes:
Name Opt Definition

railcar-type-subcategory-code OP The specific value that represents the detailed class of RAILCAR-TYPE.
