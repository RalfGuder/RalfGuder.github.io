Entity
Name: RAILWAY
Definition: A FACILITY that is a track or set of tracks made of steel rails along which trains run.
Supertype: FACILITY
Attributes:
Name Opt Definition
railway-id (PK) (FK) MA The facility-id of a specific RAILWAY (a role name for object-item-id).
railway-block-distance-dimension OP The one-dimensional linear distance representing the minimum length of the passing track intervals for single track lines or the safety margin related to the signalling system on double or multiple track lines.
railway-gross-trailing-load-quantity OP The numeric value that represents the maximum tonnage that can be moved under optimum conditions. The unit of measure is metric ton.
railway-maximum-speed-rate OP The numeric value that denotes the maximum speed that it is possible to move on a specific RAILWAY. The unit of measure is kilometres per hour. The specified value must be greater than or equal to 0 (zero).
railway-signal-system-code OP The specific value that represents the type of signal system used for the RAILWAY.
railway-signal-system-efficiency-code OP The specific value that represents the percentage value used to compute the inevitable delays caused by the type of signalling in use on the RAILWAY line.
railway-sleeper-density-dimension OP The one-dimensional linear distance representing the average horizontal distance measured from side to side and perpendicular to the central axis of the gap between two RAILWAY sleepers (ties).
railway-track-count OP The integer value representing the number of tracks on a RAILWAY line.
railway-track-gauge-code OP The specific value that represents the distance between the internal sides of rails on a RAILWAY line.
railway-traction-system-code OP The specific value that represents the motive power (engine type) that is supported along a specific RAILWAY.
railway-train-density-count OP The integer value representing the maximum number of trains, which can be moved in each direction over a specified section of track in a 24 hour period.
