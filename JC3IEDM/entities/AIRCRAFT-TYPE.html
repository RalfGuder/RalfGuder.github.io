Entity
Name: AIRCRAFT-TYPE
Definition: An EQUIPMENT-TYPE that is designed to fly.
Supertype: EQUIPMENT-TYPE
Attributes:
Name Opt Definition
aircraft-type-id (PK) (FK) MA The equipment-type-id of a specific AIRCRAFT-TYPE (a role name for object-type-id).
aircraft-type-airframe-design-code OP The specific value that represents the design of the airframe of an AIRCRAFT-TYPE.
aircraft-type-category-code OP The specific value that represents the class of AIRCRAFT-TYPE.
aircraft-type-design-range-code OP The specific value that represents the designed range of an AIRCRAFT-TYPE.
aircraft-type-design-role-code OP The specific value that represents the designed role of an AIRCRAFT-TYPE.
aircraft-type-load-category-code OP The specific value that represents a loading capability of an AIRCRAFT-TYPE.
aircraft-type-main-purpose-code OP The specific value that represents the main purpose of an AIRCRAFT-TYPE.
aircraft-type-manning-code OP The specific value that represents whether an aircraft is designed to be manned or unmanned.
aircraft-type-military-civilian-code OP The specific value that represents whether an aircraft is primarily intended for military or civilian use.
aircraft-type-model-code OP The specific value that represents the specific design of an AIRCRAFT-TYPE.
aircraft-type-takeoff-and-landing-code OP The specific value that represents the takeoff and landing designation of an AIRCRAFT-TYPE.
aircraft-type-training-category-code OP The specific value that denotes whether an aircraft can be used for training purposes.
aircraft-type-weather-qualifier-code OP The specific value that represents the weather conditions in which an AIRCRAFT-TYPE can perform its mission.
aircraft-type-wing-span-dimension OP The one-dimensional linear distance representing the spread of the wings of a specific AIRCRAFT-TYPE measured from end to end.
