Name: NETWORK-FREQUENCY

Attributes:
Name Opt Definition
network-frequency-index (PK) MA The unique value, or set of characters, assigned to represent a specific NETWORK-FREQUENCY for a specific NETWORK and to distinguish it from all other NETWORK-FREQUENCYs for that NETWORK.
network-id (PK) (FK) MA The facility-id of a specific NETWORK (a role name for object-item-id).
network-frequency-band-code OP The specific value that represents the frequency band of a specific NETWORK.
network-frequency-band-lower-frequency-quantity OP The numeric value that represents the lowest radio frequency of a specific contiguous band that a NETWORK may use. The unit of measure is kilohertz.
network-frequency-band-upper-frequency-quantity OP The numeric value that represents the highest radio frequency of a specific contiguous band that a NETWORK may use. The unit of measure is kilohertz.
network-frequency-channel-number-text OP The character string assigned to represent the channel number associated with a specific NETWORK-FREQUENCY.
network-frequency-discrete-frequency-quantity OP The numeric value that represents the radio frequency that a specific NETWORK may use. The unit of measure is kilohertz.
network-frequency-effective-end-datetime OP The character string representing a point in time that designates the end of the authorised period of effectiveness for a specific NETWORK-FREQUENCY.
network-frequency-effective-start-datetime OP The character string representing a point in time that designates the effective assignment of a specific OBJECT-TYPE-ESTABLISHMENT to a specific OBJECT-ITEM.
network-frequency-modulation-code OP The specific value that represents the modulation of a specific NETWORK.
network-frequency-purpose-text OP The character string assigned to represent the specific purpose of a NETWORK-FREQUENCY.
