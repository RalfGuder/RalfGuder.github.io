Entity
Name: MATERIEL
Definition: An OBJECT-ITEM that is equipment, apparatus or supplies of military interest without distinction as to its application for administrative or combat purposes.
Supertype: OBJECT-ITEM
Subtypes: INSTRUMENT-LANDING-SYSTEM
Attributes:
Name Opt Definition
materiel-id (PK) (FK) MA The object-item-id of a specific MATERIEL (a role name for object-item-id).
materiel-category-code OP The specific value that represents the class of MATERIEL. It serves as a discriminator that partitions MATERIEL into subtypes.
materiel-hull-number-text OP The character string assigned to represent the number assigned to a specific vessel, which when associated with a specific hull type or ship type, uniquely identifies that vessel.
materiel-lot-identification-text OP The character string assigned to represent a specific production of a specific MATERIEL.
materiel-mine-requisition-case-number-text OP The character string assigned to represent the book/record keeping number to keep track of individual mines.
materiel-serial-number-identification-text OP The character string assigned to represent a specific MATERIEL.
