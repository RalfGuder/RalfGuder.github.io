Welcome to the Data Model Browsing Tool

This browser representation provides an easy way to navigate the data model content without the need for either the MIP Information Resource Dictionary (MIRD) or CA's ERwin software. It includes the entire logical data model structure and all of its definitions.

As static HTML, the data model pages are created from the MS Access based MIRD database. The data model identification, including configuration control is displayed in the header of each page.
