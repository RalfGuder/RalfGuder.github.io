Domain
Name: action-task-status-progress-code
Definition: The specific value that represents the perceived appraisal of the progress of a specific ACTION-TASK.
Definition Source: ATCCIS
Allowable Values:
Value Definition Source Physical Value Identifier
Aborted The specified ACTION-TASK has been abandoned subsequent to its initiation. ATCCIS ABO 1000001
Airborne The specified ACTION-TASK is in the main phase at the time of the report. MIP-NDAG AIRBRN 1000008
Cancelled The specified ACTION-TASK was cancelled prior to its initiation. ATCCIS CANCLD 1000002
Complete The specified ACTION-TASK has been carried out and is complete. ATCCIS COM 1000003
Delayed The execution of the specified ACTION-TASK has been delayed beyond the planned starting time. MIP-NDAG DELAY 1000009
Deploy, maritime mine warfare equipment To deploy maritime mine warfare equipment from a vessel. ADatP-3 BL12.2 - Def MIP-NDAG DPLMMW 1000018
Down time, weather Hours that equipment e.g. helicopters, cannot operate due to weather conditions. ADatP-3 BL12.2, BL13.1 FFIRN/FUD 1047/17 - Def MIP-NDAG DWNTMW 1000015
In progress The specified ACTION-TASK is in the process of being carried out at the time of the report. ATCCIS IPR 1000004
No operation The planned task was not carried out during a specified period. ADatP-3 BL12.2, BL13.1 FFIRN/FUD 1134/3 - Def MIP-NDAG NOOPER 1000017
Not started The specified ACTION-TASK has not yet begun at the time of the report. ATCCIS NST 1000005
Off task Hours that a unit is scheduled off task. ADatP-3 BL12.2, BL13.1 FFIRN/FUD 1047/15 - Def MIP-NDAG OFFTSK 1000014
Paused The specified ACTION-TASK has been temporarily halted for an unspecified period of time. ATCCIS PAU 1000006
Planning The specified ACTION-TASK is still in the planning phase. MIP-NDAG PLAN 1000010
Recover, maritime mine warfare equipment To recover maritime mine warfare equipment to a vessel. ADatP-3 BL12.2 - Def MIP-NDAG RECMMW 1000019
Retasked The specified ACTION-TASK has been replaced or modified subsequent to its initiation. MIP-NDAG RETASK 1000011
Scrambled The order to take-off has been received by the aircraft tasked with the specified ACTION-TASK, but the action area is not yet reached. MIP-NDAG SCRMBL 1000012
Tasked The specified ACTION-TASK has been assigned, but the execution has not started. MIP-NDAG TASKED 1000013
Turning, maritime mine warfare The method of completing the end of a run on one track and preparing to commence the next run either on the same track or another track. ADatP-3 BL12.2 - Def adapted ATP-6 (c) Vol 1 TRNMMW 1000016
Not known It is not possible to determine which value is most applicable. ATCCIS NKN 1000007
Not otherwise specified The appropriate value is not in the set of specified values. ATCCIS NOS 1000020
