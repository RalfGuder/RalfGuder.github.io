Name: action-objective-item-marking-method-code
Definition: The specific value that represents the method of marking a position.
Definition Source: ATCCIS
Allowable Values:
Value Definition Source Physical Value Identifier
Flare The marking of an object by a pyrotechnic munition producing illumination or identification. Adapted OED FLR 1000001
Illumination The act or process of marking of an object by visible or near visible light. Adapted OED ILLUMN 1000003
Laser The marking of an object by a device that produces an electromagnetic beam for illumination or identification. Adapted OED LAS 1000004
Light The marking of an object by a device that places visible light for illumination or identification. Adapted OED LIGHT 1000002
Marker panel A sheet of material that is used as a reference point for object marking. Adapted JCS Pub 1-02 MPL 1000005
Radio beacon The reference point of an object by a radio transmitter that emits a distinctive, or characteristic, signal used for the determination of bearings, courses or location. Adapted JCS Pub 1-02 RBE 1000006
Smoke The marking of an object by a munition producing smoke. Adapted OED SMOKE 1000007
Star cluster The marking of an object by use of a star cluster for illumination or identification. ADatP-3 BL13.1 FFIRN/FUD 1613/1 - Def MIP STRCLS 1000011
Strobe beacon The marking of an object by use of a strobe beacon for illumination or identification. ADatP-3 BL13.1 FFIRN/FUD 1613/1 - Def MIP STRBBN 1000012
Strobe lights The marking of an object by use of strobe lights for illumination or identification. ADatP-3 BL13.1 FFIRN/FUD 1613/1 - Def MIP STRBLT 1000013
Torch The marking of an object by use of a torch for illumination or identification. ADatP-3 BL13.1 FFIRN/FUD 1613/1 - Def MIP TORCH 1000014
Vehicle lights The marking of an object by lights from a vehicle for illumination or identification. ADatP-3 BL13.1 FFIRN/FUD 1613/1 - Def MIP VHCLLT 1000015
Not known It is not possible to determine which value is most applicable. ATCCIS NKN 1000008
Not otherwise specified The appropriate value is not in the set of specified values. ATCCIS NOS 1000010
