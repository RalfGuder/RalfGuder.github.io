Domain
Name: action-task-start-qualifier-code
Definition: The specific value that denotes the role of starting date and time with respect to the period of effectiveness of a specific ACTION-TASK.
Definition Source: ATCCIS
Allowable Values:
Value Definition Source Physical Value Identifier
After Time intended is later than the time specified. ATCCIS AFT 1000001
As soon as possible Begin the activity at the earliest possible time once execution is authorised. ATCCIS ASAP 1000006
As soon as possible after Begin the activity at the earliest possible time after the specified start time. ATCCIS ASAPAF 1000007
As soon as possible not later than Begin the activity at the earliest possible time but not later than the specified start time. ATCCIS ASAPNL 1000008
At Time intended is the time specified. ATCCIS AT 1000002
Before Time intended is in advance of the time specified. ATCCIS BEF 1000003
No later than Time specified is the latest. ATCCIS NLT 1000004
Not before Time specified is the earliest. ATCCIS NOB 1000005
On call Time will be specified later by a dedicated call. ADatP-3 BL12, BL13.1 FFIRN/FUD 1220/8 - Def MIP-NDAG ONCALL 1000009
On codeword Begin the activity on receipt of codeword. MIP-NDAG - Def adapted OED ONCDWD 1000012
To be determined Time intended is to be determined later. ADatP-3 BL12, BL13.1 FFIRN/FUD 1220/8 - Def MIP-NDAG TBD 1000010
Unknown Time intended is not known. ADatP-3 BL12, BL13.1 FFIRN/FUD 1220/8 - Def MIP-NDAG UNK 1000011
