Name: reference-content-category-code
Definition: The specific value that represents the classification of the general content of the artefact cited in a specific REFERENCE.
Definition Source: MIP-NDAG
Allowable Values:
Value Definition Source Physical Value Identifier
Administrative document The artefact cited in the specific REFERENCE provides administrative information. MIP-NDAG ADMDOC 1000001
Authorisation The artefact cited in the specific REFERENCE constitutes an authorising document. MIP-NDAG AUTHRS 1000002
Certificate The artefact cited in the specific REFERENCE is an official document authorised for a specific use. MIP-NDAG CERTFC 1000003
Directive The artefact cited in the specific REFERENCE provides compulsory instructions. MIP-NDAG DRCTV 1000004
Guidance The artefact cited in the specific REFERENCE provides advisory information. MIP-NDAG GUIDNC 1000005
Health document The artefact cited in the specific REFERENCE records medical information. MIP-NDAG HLTDOC 1000006
Historical document The artefact cited in the specific REFERENCE records historical information. MIP-NDAG HISDOC 1000007
Identification document The artefact cited in the specific REFERENCE is an official document to ascertain identity. MIP-NDAG IDDOC 1000008
Instruction The artefact cited in the specific REFERENCE provides information to support learning. MIP-NDAG INSTRC 1000009
Legal document The artefact cited in the specific REFERENCE records information regarding legal matters. MIP-NDAG LGLDOC 1000010
Map The artefact cited in the specific REFERENCE provides graphic geographic information. MIP-NDAG MAP 1000011
Message text format The artefact cited in the specific REFERENCE is composed of several sets ordered in a specified sequence, each set characterized by an identifier and containing information of a specified type, coded and arranged in an ordered sequence of character fields in accordance with the NATO message text formatting rules. MIP-NDAG - Def adapted AAP-6 MTF 1000020
Order The artefact cited in the specific REFERENCE is a document that is directive in nature and generally used within the military sphere. MIP-NDAG ORDER 1000012
Personnel document The artefact cited in the specific REFERENCE records information regarding matters affecting personnel. MIP-NDAG PERDOC 1000013
Report The artefact cited in the specific REFERENCE is an authoritative account of an event or situation. MIP-NDAG REPORT 1000014
Staff journal/log The artefact cited in the specific REFERENCE is a detailed account of the activities of a staff. MIP-NDAG STFFJR 1000015
Technical document The artefact cited in the specific REFERENCE provides technical description. MIP-NDAG TECDOC 1000016
Template The artefact cited in the specific REFERENCE provides a pre-defined guide. MIP-NDAG TEMPLT 1000017
Test document The artefact cited in the specific REFERENCE provides information about test design or test results. MIP-NDAG TSTDOC 1000018
Not otherwise specified The appropriate value is not in the set of specified values. MIP-NDAG NOS 1000019
