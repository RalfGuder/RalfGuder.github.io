Name: action-task-status-amend-timing-code
Definition: The specific value that denotes request or requirement to modify the timing associated with a specific ACTION-TASK.
Definition Source: ATCCIS
Allowable Values:
Value Definition Source Physical Value Identifier
Request end amend time The times entered into this entity represent the reporting organisation's requested alterations to the ACTION-TASK timings. ATCCIS RQSEAT 1000001
Request start amend time The times entered into this entity represent the reporting organisation's requested alterations to the ACTION-TASK timings. ATCCIS RQSSAT 1000003
Require end amend time The timings entered into this entity represent the new timings the reporting organisation requires to achieve its part of the ACTION-TASK. ATCCIS RQEEAT 1000002
Require start amend time The timings entered into this entity represent the new timings the reporting organisation requires to achieve its part of the ACTION-TASK. ATCCIS RQESAT 1000004
