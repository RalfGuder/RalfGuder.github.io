Domain
Name: minefield-maritime-bottom-natural-camouflage-code
Definition: The specific value that represents the description of the ground of a lake, river, or ocean, or other body of water with respect to the ability to hide mines on the bottom at a specific MINEFIELD-MARITIME.
Definition Source: ADatP-3 BL13.1 FFIRN/FUD 1423/3 and UK MWC adapted
Allowable Values:
Value Definition Source Physical Value Identifier
Bottom covered in seaweed Mines are likely to be hidden by seaweed, mine hunting possible with difficulty. ADatP-3 BL12.2 1423/2, BL13.1 FFIRN/FUD 1423/3 - Def UK MWC BTMCVR 1000007
Bottom has deep hollows or crevices or cliffs Mines are likely to be hidden completely in deep hollows or crevices or by cliffs, mine hunting difficult. ADatP-3 BL12.2 1423/2, BL13.1 FFIRN/FUD 1423/3 - Def UK MWC BTMDEP 1000005
Irregular bottom Mines are likely to be hidden by irregularity of the bottom, mine hunting difficult. ADatP-3 BL12.2 1423/2, BL13.1 FFIRN/FUD 1423/3 - Def UK MWC IRREGL 1000006
Rather stable and smooth but uneven bottom Rather stable and smooth but uneven bottom, mine hunting possible. ADatP-3 BL12.2 1423/2, BL13.1 FFIRN/FUD 1423/3 - Def UK MWC RTHRST 1000003
Rough bottom Rough bottom, mine hunting more difficult. ADatP-3 BL12.2 1423/2, BL13.1 FFIRN/FUD 1423/3 - Def UK MWC RGHBTM 1000002
Soft bottom Mines are likely to be hidden completely owing to complete burial, mine hunting difficult. ADatP-3 BL12.2 1423/2, BL13.1 FFIRN/FUD 1423/3 - Def UK MWC SFTBTM 1000004
Stable and smooth flat bottom Stable and smooth flat bottom, mine hunting possible, little camouflage for mines. ADatP-3 BL12.2 1423/2, BL13.1 FFIRN/FUD 1423/3 - Def UK MWC STBSMT 1000001
Not otherwise specified The appropriate value is not in the set of specified values. ATCCIS NOS 1000008
