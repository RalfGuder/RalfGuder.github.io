Domain
Name: geographic-feature-bottom-hardness-code
Definition: A specific value that represents the subjective indication obtained by a diver of the hardness of the liquid/solid surface interface and is the result of a single arm thrust.
Definition Source: ADatP-3 BL13.1 FFIRN 1184/2 adapted
Allowable Values:
Value Definition Source Physical Value Identifier
Arm penetrates to elbow The depth to which the divers arm penetrates the surface bottom, using a clenched fist, up to the elbow. ADatP-3 BL12.2, BL13.1 FFIRN/FUD 1184/2 - Def AML V 2.1 NOV 05 adapted ARMELB 1000001
Arm penetrates to shoulder The depth to which the divers arm penetrates the surface bottom, using a clenched fist, up to the shoulder. ADatP-3 BL12.2, BL13.1 FFIRN/FUD 1184/2 - Def AML V 2.1 NOV 05 adapted ARMSHO 1000004
Arm penetrates to wrist The depth to which the divers arm penetrates the surface bottom, using a clenched fist, up to the wrist. ADatP-3 BL12.2, BL13.1 FFIRN/FUD 1184/2 - Def AML V 2.1 NOV 05 adapted ARMWRS 1000005
Hand penetrates to knuckles The depth to which the divers arm penetrates the surface bottom, using extended fingers, up to the knuckles. ADatP-3 BL12.2, BL13.1 FFIRN/FUD 1184/2 - Def AML V 2.1 NOV 05 adapted ARMKNU 1000002
Hand penetrates to palm The depth to which the divers arm penetrates the surface bottom, using extended fingers, up to the palm. ADatP-3 BL12.2, BL13.1 FFIRN/FUD 1184/2 - Def AML V 2.1 NOV 05 adapted ARMPLM 1000003
No penetration The depth to which the divers arm penetrates the surface bottom, using a clenched fist, resulting in no penetration. ADatP-3 BL12.2, BL13.1 FFIRN/FUD 1184/2 - Def AML V 2.1 NOV 05 adapted NOPENT 1000006
