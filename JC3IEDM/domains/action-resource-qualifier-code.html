Domain
Name: action-resource-qualifier-code
Definition: The specific value that represents the type of restriction or other qualification applicable to a specific ACTION-RESOURCE for a specific ACTION.
Definition Source: ATCCIS
Allowable Values:
Value Definition Source Physical Value Identifier
Authorised for use The specified ACTION-RESOURCE is authorised for use without restriction. ATCCIS AUTH 1000001
Maximise use of The specified ACTION-RESOURCE is authorised and its use is to be maximised. ATCCIS MAXU 1000002
Minimise use of The specified ACTION-RESOURCE is authorised, but use of it is to be minimised to only that which is operationally necessary in the opinion of the authorised commander. ATCCIS MINU 1000003
No exploitation east of line The specific ACTION-RESOURCE is authorised with the restriction that there is to be no exploitation east of it. MIP NEEL 1000004
No exploitation north of line The specific ACTION-RESOURCE is authorised with the restriction that there is to be no exploitation north of it. MIP NENL 1000005
No exploitation south of line The specific ACTION-RESOURCE is authorised with the restriction that there is to be no exploitation south of it. MIP NESL 1000006
No exploitation west of line The specific ACTION-RESOURCE is authorised with the restriction that there is to be no exploitation west of it. MIP NEWL 1000007
Not authorised The specified ACTION-RESOURCE is not authorised for use. ATCCIS NOTA 1000009
Stay above The specific ACTION-RESOURCE is authorised with the restriction that there is to be no movement below or within it. MIP-NDAG STAYAB 1000010
Stay below The specific ACTION-RESOURCE is authorised with the restriction that there is to be no movement above or within it. MIP-NDAG STAYBL 1000011
Stay inside The specific ACTION-RESOURCE is authorised with the restriction that there is to be no movement outside it. MIP STAYIN 1000012
Stay outside The specific ACTION-RESOURCE is authorised with the restriction that there is to be no movement within it. MIP STAYOT 1000013

Entity Attribute Opt
