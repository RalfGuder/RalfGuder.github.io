Domain
Name: ied-tactical-characterization-employment-method-code
Definition: The specific value that describes how the device is delivered or attached to the target.
Definition Source: WTI IED Lexicon DEC 2008 adapted
Allowable Values:
Value Definition Source Physical Value Identifier
Air-borne An IED held aloft by aerodynamic means or buoyancy that serves as concealment means for explosives with an initiating device. WTI IED Lexicon DEC 2008 - Def WTI IED Lexicon DEC 2008 adapted ABIED 1000001
Animal-borne An IED worn by an animal where the whole IED or principal IED components are placed on the animal and also serves as the delivery or concealment means for explosives with an initiating device. MIP OWG - Def MIP-DMSWG JAN 2010 ANIED 1000002
Large vehicle-borne An IED built into any large ground-based vehicle (e.g., dump truck, panel truck, bongo truck, commercial bus, tanker, etc.) that serves as the concealment means for explosives with an initiating device. WTI IED Lexicon DEC 2008 - Def WTI Handbook SEP 2009 LVBIED 1000003
Magnetic attachment A type of IED employment in which the device is attached to the target using magnets. WTI IED Lexicon DEC 2008 - Def WTI IED Lexicon DEC 2008 TACHMG 1000004
Non-magnetic attachment A type of IED employment in which the device is attached to the target using non-magnetic means. WTI IED Lexicon DEC 2008 - Def WTI IED Lexicon DEC 2008 TACHNM 1000005
Person-borne An IED worn by a person where the whole IED or principal IED components are placed in a vest, belt, backpack, etc, and also serves as the delivery or concealment means for explosives with an initiating device. WTI IED Lexicon DEC 2008 adapted - Def WTI IED Lexicon DEC 2008 adapted PBIED 1000006
Postal/Package delivery An IED introduced or delivered through a postal or package delivery system. WTI IED Lexicon DEC 2008 - Def WTI IED Lexicon DEC 2008 adapted POSTAL 1000007
Projected An IED delivered by an improvised weapons system that delivers the main charge through the air to its target. WTI IED Lexicon DEC 2008 - Def WTI IED Lexicon DEC 2008 adapted PROJD 1000008
Underbelly A type of IED employment in which the device targets the underside of a vehicle, using large amounts of explosives buried to deliberately defeat armour (can include conventional land mines). WTI IED Lexicon DEC 2008 - Def WTI Handbook SEP 2009 UBELLY 1000009
Vehicle-borne An IED delivered by any small ground-based vehicle (e.g., passenger vehicle, motorcycle, moped, bicycle, etc.) that serves as the concealment means for explosives with an Initiating device. WTI IED Lexicon DEC 2008 - Def WTI Handbook SEP 2009 VBIED 1000010
Vessel-borne An IED delivered by any vessel that serves as the concealment means for explosives with an Initiating device. MIP-OWG JAN 2010 - Def MIP-DMSWG JAN 2010 VSBIED 1000011
Water-borne An IED delivered by floating, drifting, anchored, or propelled on or below the water that serves as the concealment means for explosives with an initiating device. WTI IED Lexicon DEC 2008 - Def WTI Handbook SEP 2009 WBIED 1000012
Not known It is not possible to determine which value is most applicable. ATCCIS NKN 1000014
Not otherwise specified The appropriate value is not in the set of specified values. ATCCIS NOS 1000013
