Domain
Name: object-type-category-code
Definition: The specific value that represents the class of OBJECT-TYPE.
Definition Source: ATCCIS
Allowable Values:
Value Definition Source Physical Value Identifier
FACILITY-TYPE An OBJECT-TYPE that is intended to be built, installed or established to serve some particular purpose and is identified by the service it is intended to provided rather than by its content. Adapted JCS Pub 1-02 FA 1000001
FEATURE-TYPE An OBJECT-TYPE that encompasses meteorological, geographic, and control features of military significance. ATCCIS FE 1000002
MATERIEL-TYPE An OBJECT-TYPE that represents equipment, apparatus or supplies of military interest without distinction to its application for administrative or combat purposes. Adapted JCS Pub 1-02 MA 1000003
ORGANISATION-TYPE An OBJECT-TYPE that represents administrative or functional structures. Adapted JCS Pub 1-02 OR 1000004
PERSON-TYPE An OBJECT-TYPE that represents human beings about whom information is to be held. Adapted JCS Pub 1-02 PE 1000005
Not known It is not possible to determine which value is most applicable. ATCCIS NKN 1000006
