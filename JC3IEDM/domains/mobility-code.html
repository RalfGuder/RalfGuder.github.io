Domain
Name: mobility-code
Definition: The specific value that indicates the suitability of a specific ROUTE or ROUTE-SEGMENT for movement.
Definition Source: MIP-NDAG
Allowable Values:
Value Definition Source Physical Value Identifier
Foot Suitable only for pedestrians and animals. ADatP-3 BL13.1 FFIRN/FUD 1269/4 - Def MIP-NDAG FOOT 1000001
Tracked Suitable for tracked vehicles. ADatP-3 BL13.1 FFIRN/FUD 1269/4 adapted - Def MIP-NDAG TRACK 1000003
Wheeled and tracked Suitable for wheeled and tracked vehicles. ATCCIS WHTR 1000004
Wheeled, all road Suitable for wheeled vehicles with improved capabilities. MIP WHLAWD 1000007
Wheeled, general Suitable for wheeled vehicles without need for any additional capability. ADatP-3 BL13.1 FFIRN/FUD 1269/4 adapted - Def MIP-NDAG WHEEL 1000002
Not known It is not possible to determine which value is most applicable. ATCCIS NKN 1000006
Usage:
Entity Attribute Opt
ROUTE route-mobility-code OP
ROUTE-SEGMENT route-segment-mobility-code OP
