Name: candidate-target-detail-authorisation-approval-code
Definition: The specific value that represents the type of approval that a specific CANDIDATE-TARGET-DETAIL is authorised for further consideration in planning military operations.
Definition Source: ATCCIS
Allowable Values:
Value Definition Source Physical Value Identifier
Approved and available The specific CANDIDATE-TARGET-DETAIL is authorised for engagement. ATCCIS APPAVL 1000001

Not approved The specific CANDIDATE-TARGET-DETAIL is not to be used in planning. ATCCIS NOTAPP 1000003
