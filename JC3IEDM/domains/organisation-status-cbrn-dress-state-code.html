Domain
Name: organisation-status-cbrn-dress-state-code
Definition: The specific value that represents the Mission Oriented Protective Posture (MOPP) status defining the NBC (CBRN) dress code of a specific ORGANISATION.
Definition Source: CBRN COI
Allowable Values:
Value Definition Source Physical Value Identifier
MOPP 0 Mask is carried. Individual protection equipment is immediately available. QSTAG 1045 (to become STANAG 2984) MOPP0 1000002
MOPP 1 Mask is carried. Suit worn, boots and gloves are carried. QSTAG 1045 (to become STANAG 2984) MOPP1 1000003
MOPP 2 Mask is carried. Suit and boots worn, gloves are carried. QSTAG 1045 (to become STANAG 2984) MOPP2 1000004
MOPP 3 Mask is carried. Suit, boots and gloves are worn. QSTAG 1045 (to become STANAG 2984) MOPP3 1000005
MOPP Ready Mask is carried. First set individual protection equipment is available within 2 hours, second set individual protection equipment is available within 6 hours. QSTAG 1045 (to become STANAG 2984) MOPPRY 1000001
