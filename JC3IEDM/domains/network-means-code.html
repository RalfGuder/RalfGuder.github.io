Domain
Name: network-means-code
Definition: The specific value that represents the physical linkage between the nodes of a specific NETWORK.
Definition Source: MIP-NDAG
Allowable Values:
Value Definition Source Physical Value Identifier
Cable A transmission means using electrical signals, typically through copper cable. NDAG CABLE 1000001
Coaxial cable A transmission means consisting of a centre conductor surrounded by an insulating material and a concentric outer conductor. MIP COAXCB 1000011
Dedicated line The NETWORK is set up through a reserved public switched telephone network circuit. MIP DDCDLN 1000009
Dial-up The NETWORK is set up through a public switched telephone network. MIP DIALUP 1000010
Fibre optic A transmission means using light transmission using fibre optic cable. NDAG FBROPT 1000002
Infrared A transmission means using infrared light. ADat-P3 BL12, BL13.1 FFIRN/FUD 1275/1 - Def MIP-NDAG INFRED 1000015
Laser A transmission means using a laser beam. ADat-P3 BL12, BL13.1 FFIRN/FUD 1275/1 - Def MIP-NDAG LASER 1000016
Maser A transmission means using a maser beam (amplified electromagnetic waves). ADat-P3 BL12, BL13.1 FFIRN/FUD 1275/1 - Def MIP-NDAG MASER 1000017
Microwave A transmission means using micrometric frequency electromagnetic waves. ADat-P3 BL12, BL13.1 FFIRN/FUD 1275/1 - Def MIP-NDAG MICROW 1000018
Mixed A transmission where more than one type of network means is used across a single communication link. NDAG MIXED 1000003
Radio link, general A transmission means using the Electromagnetic spectrum. NDAG RDLNKG 1000004
Radio link, satellite A transmission means using the Electromagnetic spectrum specifically through a satellite. NDAG RDLNKS 1000005
Radio link, terrestrial A transmission means using the Electromagnetic spectrum specifically through a path that does not include a satellite. MIP RDLTER 1000013
Radio link, tropospheric A transmission means using the Electromagnetic spectrum and specifically the reflective properties of the troposphere. ADat-P3 BL11, BL13.1 FFIRN/FUD 1175/2 adapted - Def MIP-NDAG RDLTRO 1000014
Radio relay A transmission means using a highly directional line of sight transmission mainly for trunk communications. NDAG RDRLY 1000006
Twisted pair cable A transmission means made up of one or more separately insulated twisted-wire pairs, none of which is arranged with another to form quads. MIP TWTPRC 1000012
Not known It is not possible to determine which value is most applicable. ATCCIS NKN 1000007
Not otherwise specified The appropriate value is not in the set of specified values. ATCCIS NOS 1000008
