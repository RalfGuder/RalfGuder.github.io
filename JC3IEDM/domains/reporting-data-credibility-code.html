Domain
Name: reporting-data-credibility-code
Definition: The specific value that represents, for normal operational use, the degree of trustworthiness of the data referenced by a specific REPORTING-DATA.
Definition Source: ATCCIS
Allowable Values:
Value Definition Source Physical Value Identifier
Indeterminate Basis for the estimate cannot be ascertained. Adapted OED IND 1000001
Reported as a fact Data is reported by different sources whose integrity is not in question. Adapted OED RPTFCT 1000005
Reported as plausible Reported data is considered as possible or probable. Adapted OED RPTPLA 1000006
Reported as uncertain Reported data is open to or can be viewed with suspicion. Adapted OED RPTUNC 1000007
Usage:
Entity Attribute Opt
REPORTING-DATA reporting-data-credibility-code OP
