Name: minefield-maritime-status-colour-code
Definition: The specific value that represents the known status of mined channels of a MINEFIELD-MARITIME.
Definition Source: ADatP-3 BL13.1 FFIRN/FUD 1138/8 adapted and UK MWTU
Allowable Values:
Value Definition Source Physical Value Identifier
Green Established channel/RTE where information does not exist on enemy mining or all known mines in the channel have been countered. ADatP-3 BL12.2 1145/2, BL13.1 FFIRN/FUD 1138/8 - Def BL13.1 FFIRN/FUD 1138/8 adapted GREEN 1000001
Red Channel where mines are known to be present. ADatP-3 BL12.2 1145/2, BL13.1 FFIRN/FUD 1138/8 - Def BL13.1 FFIRN/FUD 1138/8 adapted RED 1000002
Yellow Channel where the degree of danger has been reduced by mine counter measure operations. ADatP-3 BL12.2 1145/2, BL13.1 FFIRN/FUD 1138/8 - Def BL13.1 FFIRN/FUD 1138/8 adapted YELLOW 1000003
