Name: organisation-status-reinforcement-code
Definition: The specific value that represents whether a specific ORGANISATION has additional or detached strength.
Definition Source: ATCCIS
Allowable Values:
Value Definition Source Physical Value Identifier
Detached only The UNIT in question has sub-UNITs detached. ATCCIS DETD 1000001
Normal strength The UNIT in question has no attachments or detachments. ATCCIS NORM 1000002
Reinforced and detached The UNIT in question has UNITs attached and sub-UNITs detached. ATCCIS RIDT 1000003
Reinforced only The UNIT in question has additional UNITs attached. ATCCIS REIN 1000004
Not known It is not possible to determine which value is most applicable. ATCCIS NKN 1000005
