Domain
Name: order-category-code
Definition: The specific value that represents the type of ORDER.
Definition Source: MIP-NDAG
Allowable Values:
Value Definition Source Physical Value Identifier
Administrative/Logistics order An order to give the commander's plan for administrative and service support of operations. STANAG 2014 Ed 9 ADLOGO 1000001
Fragmentary order to existing operation order An abbreviated form of an Operations order. Adapted STANAG 2014 Ed 9 FRAGO 1000002
Operations order An order that includes only such detail as is necessary for commanders of subordinate formations/units to issue their own orders and to ensure coordination. STANAG 2014 Ed 9 OPORD 1000003
Warning order An order to assist units and their staffs initiate the preparations for and the execution of their new mission by giving them the maximum warning and essential details of impending operations and information of time available. Adapted STANAG 2014 Ed 9 WNGO 1000004
Not otherwise specified The appropriate value is not in the set of specified values. ATCCIS NOS 1000005
