Domain
Name: facility-status-security-status-code
Definition: The specific value that represents the security status of a specific FACILITY.
Definition Source: ATCCIS
Allowable Values:
Value Definition Source Physical Value Identifier
Guarded The specific facility is being guarded so as to control entry or exit. ATCCIS GUARDD 1000001
None No special action has been taken to guarantee the security of the specific facility. ATCCIS NONE 1000002
Secured The specific facility is protected or safe. ATCCIS SECURD 1000003
Not known It is not possible to determine which value is most applicable. ATCCIS NKN 1000004

Entity Attribute Opt
