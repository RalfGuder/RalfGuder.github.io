Domain
Name: handling-capability-descriptor-code
Definition: The specific value that represents the HANDLING-CAPABILITY that is being quantified.
Definition Source: ACT
Allowable Values:
Value Definition Source Physical Value Identifier
Bulk liquid The numeric value representing the maximum amount of any unpackaged liquid. MIP-NDAG BLKLIQ 1000001
Bulk volume The numeric value representing the maximum volume of any unpackaged mass. MIP-NDAG BLKVOL 1000002
Maximum cargo height The one-dimensional linear measurement that represents the extreme vertical distance, measured from the lowest to the highest reference point, of any object. MIP-NDAG MCRHEI 1000003
Maximum cargo length The one-dimensional linear measurement that represents the extreme horizontal distance, measured from side to side and perpendicular to the central axis, of any object. MIP-NDAG MCRLEN 1000004
Maximum cargo weight The numeric value representing the maximum weight of any cargo. MIP-NDAG MCRWGT 1000005
Maximum cargo width The one-dimensional linear measurement that represents the extreme horizontal distance, measured from side to side and parallel to the central axis, of any object. MIP-NDAG MCRWID 1000006
Maximum count The numeric value representing the maximum item count. ACT MAXCNT 1000007
NEQ limit The numeric value representing the Net Explosive Quantity limit equivalent to TNT (trinitrotoluene) explosive power. LOGBASE NEQLMT 1000008
