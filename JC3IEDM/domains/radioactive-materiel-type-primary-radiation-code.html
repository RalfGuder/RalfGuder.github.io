Domain
Name: radioactive-materiel-type-primary-radiation-code
Definition: The specific value that represents the most intense radiation emitted by a RADIOACTIVE-MATERIEL-TYPE.
Definition Source: AEP/ATP-45(B)
Allowable Values:
Value Definition Source Physical Value Identifier
Alpha radiation Ionising radiation consisting of alpha particles (a helium nucleus), emitted by some substances undergoing radioactive decay. COD ALPHA 1000001
Beta radiation Radioactive decay in which an electron is emitted. COD BETA 1000002
Gamma radiation Penetrating electromagnetic radiation of shorter wavelength than X-rays. COD GAMMA 1000003
Neutron Subatomic particle of about the same mass as a proton but without an electric charge, present in all atomic nuclei except those of ordinary hydrogen emitted during nuclear decay. COD NEUTRN 1000004
Not known It is not possible to determine which value is most applicable. ATCCIS NKN 1000005

Entity Attribute Opt
