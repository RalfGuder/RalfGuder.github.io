Name: railcar-type-category-code
Definition: The specific value that represents the class of RAILCAR-TYPE.
Definition Source: ATCCIS
Allowable Values:
Value Definition Source Physical Value Identifier
Locomotive A detachable, wheeled engine used for pulling trains. Adapted OCBRH LCMTVE 1000001
Railed equipment Mobile equipment that uses rails to move on, for example, dockyard cranes. Adapted OCBRH RLDEQP 1000002
Rolling stock Generic term for wagons, either passenger, freight or specialised that are used to form a train. Adapted OCBRH RLLSTK 1000003
Train A self-propelled passenger carrying vehicle that runs on rails. ADatP-3 BL12, BL13.1 FFIRN/FUD 1650/6 adapted - Def MIP-NDAG TRAIN 1000009
Tram A passenger carrying vehicle that runs on rails normally along roads, with minimal earthworks, typically powered by electricity from overhead power cables via a pantograph. Adapted OCBRH TRAM 1000004
Not known It is not possible to determine which value is most applicable. ATCCIS NKN 1000005
Not otherwise specified The appropriate value is not in the set of specified values. ATCCIS NOS 1000006
