Name: object-type-establishment-environment-condition-code
Definition: The specific value that represents the environmental conditions for which a specific OBJECT-TYPE-ESTABLISHMENT is authorised.
Definition Source: MIP
Allowable Values:
Value Definition Source Physical Value Identifier
Arctic An indication that the specified establishment is authorised for use in arctic conditions. ATCCIS ARC 1000001
Desert An indication that the specified establishment is authorised for use in desert conditions. ATCCIS DES 1000002
Jungle An indication that the specified establishment is authorised for use in jungle conditions. ATCCIS JUN 1000003
Mountain An indication that the specified establishment is authorised for use in mountain conditions. ATCCIS MOUNTN 1000004
Temperate An indication that the specified establishment is authorised for use in temperate conditions. ATCCIS TMP 1000005
Tropical An indication that the specified establishment is authorised for use in tropical conditions. ATCCIS TRP 1000006
Not known It is not possible to determine which value is most applicable. ATCCIS NKN 1000007
Not otherwise specified The appropriate value is not in the set of specified values. ATCCIS NOS 1000008
