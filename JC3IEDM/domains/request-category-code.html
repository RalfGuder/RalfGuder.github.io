Name: request-category-code
Definition: The specific value that represents the type classification of a specific REQUEST.
Definition Source: ATCCIS
Allowable Values:
Value Definition Source Physical Value Identifier
Action To discover the actions of an ACTION-OBJECTIVE (OBJECT-ITEM) by any means. ATCCIS ACTION 1000001
Association To discover the relation between two ACTION-OBJECTIVEs (OBJECT-ITEMs) by any means. ATCCIS ASSOC 1000002
Capability To discover the capability of an ACTION-OBJECTIVE (OBJECT-ITEM) by any means. ATCCIS CAPAB 1000003
Holding To discover the holding of an ACTION-OBJECTIVE (OBJECT-ITEM) by any means. ATCCIS HOLDNG 1000004
Hostility To recognise the friendly or enemy character of an ACTION-OBJECTIVE (OBJECT-ITEM) by any means. ATCCIS HOSTIL 1000005
Location To discover the location of an ACTION-OBJECTIVE (OBJECT-ITEM) by any means. ATCCIS LOCATN 1000006
Presence To discover the manifestation of an ACTION-OBJECTIVE in the area of operational interest specified through another ACTION-OBJECTIVE (OBJECT-ITEM: FACILITY, FEATURE). ATCCIS PRESNC 1000007
Status To recognise the status of an ACTION-OBJECTIVE (OBJECT-ITEM) by any means. ATCCIS STATUS 1000008
Type To recognise the type of an ACTION-OBJECTIVE (OBJECT-ITEM) by any means. ATCCIS TYPE 1000009
