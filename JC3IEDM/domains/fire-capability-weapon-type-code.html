Domain
Name: fire-capability-weapon-type-code
Definition: The specific value that represents the general type of weapon that an EQUIPMENT-TYPE is qualified to employ.
Definition Source: MIP-NDAG
Allowable Values:
Value Definition Source Physical Value Identifier
Conventional The capability to deliver weapons which are not nuclear. Adapted AAP-6 CONV 1000001
Dual capable The capability of executing both conventional and nuclear missions. Adapted AAP-6 DUAL 1000003
Non combat capable The inability to deliver any type of weapon. MIP-NDAG NCC 1000004
Nuclear The capability to deliver nuclear weapons. MIP-NDAG NUC 1000002
Not known It is not possible to determine which value is most applicable. ATCCIS NKN 1000005
Not otherwise specified The appropriate value is not in the set of specified values. ATCCIS NOS 1000006
Usage:
Entity Attribute Opt
