Domain
Name: orbit-area-alignment-code
Definition: The specific value that represents the placement of a specific ORBIT-AREA with respect to its defining reference axis.
Definition Source: ATCCIS
Allowable Values:
Value Definition Source Physical Value Identifier
Centre The sides of a specific ORBIT-AREA are equidistant from the reference axis. ATCCIS CENTRE 1000001
Left The right side of a specific ORBIT-AREA coincides with the reference axis. ATCCIS LEFT 1000002
Right The left side of a specific ORBIT-AREA coincides with the reference axis. ATCCIS RIGHT 1000003
Usage:
Entity Attribute Opt
ORBIT-AREA orbit-area-alignment-code OP
