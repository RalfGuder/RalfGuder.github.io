Domain
Name: railway-signal-system-efficiency-code
Definition: The specific value that represents the percentage value used to compute the inevitable delays caused by the type of signalling in use on the RAILWAY line.
Definition Source: Adapted AIntP-3 (SIGNAL SYSTEM EFFICIENCY)
Allowable Values:
Value Definition Source Physical Value Identifier
50-65% The specific efficiency percentage represents rudimental signalling. Adapted AIntP-3 50 1000001
70-75% The specific efficiency percentage represents normal block operations. Adapted AIntP-3 70 1000002
80-85% The specific efficiency percentage represents automatic signalling and/or centralised traffic control. Adapted AIntP-3 80 1000003
85% The specific efficiency percentage represents computerised traffic control. Adapted AIntP-3 85 1000004

Entity Attribute Opt
