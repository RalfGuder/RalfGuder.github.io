Domain
Name: speed-precision-code
Definition: The specific value that represents the maximum degree of discrimination or resolution for which speed is stated.
Definition Source: MIP-NDAG
Allowable Values:
Value Definition Source Physical Value Identifier
Kilometres per hour The speed is defined to the precision of kilometres per hour. MIP-NDAG KPH 1000001
Knots The speed is defined to the precision of a knot. MIP-NDAG KNOTS 1000002
Metres per second The speed is defined to the precision of metres per second. MIP-NDAG MPS 1000004
