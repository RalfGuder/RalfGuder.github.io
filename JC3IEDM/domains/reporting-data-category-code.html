Domain
Name: reporting-data-category-code
Definition: The specific value that represents, for usual operational purposes, the nature of a specific REPORTING-DATA.
Definition Source: ATCCIS
Allowable Values:
Value Definition Source Physical Value Identifier
Assumed A REPORTING-DATA that points to data that is considered to be valid without being based on fact or observation. ATCCIS ASS 1000001
Erroneous A REPORTING-DATA that points to data that is wrong. ATCCIS ERR 1000002
Inferred A REPORTING-DATA that points to data derived from multiple sources. ATCCIS INFER 1000003
Planned A REPORTING-DATA that points to data expected to be true in the future. Adapted OED PLAN 1000004
Predicted A REPORTING-DATA that points to data that results from extrapolating, estimating, or foretelling a future condition or state. Adapted OED PRDCTD 1000007
Reported A REPORTING-DATA that points to data based on fact or observation. Adapted OED REP 1000005
Usage:
Entity Attribute Opt
REPORTING-DATA reporting-data-category-code OP
