Domain
Name: action-aircraft-employment-deplanement-method-code
Definition: The specific value that represents the standard method of deplanement of a specific ACTION-RESOURCE in support of a specific ACTION.
Definition Source: ATCCIS
Allowable Values:
Value Definition Source Physical Value Identifier
Abseil To deplane using rappelling equipment while the aircraft is airborne. APP-9 - Def ATCCIS ABSEIL 1000001
Gravity To deplane cargo without a parachute from the aircraft while it is airborne by gravitational force. ADatP-3 F068 BL12, BL13.1 FFIRN/FUD 1349/1adapted - Def MIP-NDAG GRAVTY 1000007
Hover To deplane without any mechanical means from an aircraft hovering close to the ground. APP-9 - Def ATCCIS HOVER 1000002
Land To deplane without any mechanical means from a landed aircraft. APP-9 - Def ATCCIS LAND 1000003
Parachute To deplane by jumping out of the aircraft while it is airborne and deploy a parachute before reaching the ground. APP-9 - Def MIP-NDAG PARCHT 1000006
Parachute, automatic To deplane from the aircraft while it is airborne with a parachute that will unfold itself automatically after leaving the aircraft. ADatP-3 F068 BL12, BL13.1 FFIRN/FUD 1349/1adapted - Def MIP-NDAG PARAUT 1000008
Parachute, extraction To deplane from the aircraft while it is airborne with a parachute that will pull the cargo out of the aircraft, and then unfold the main parachute. ADatP-3 F068 BL12, BL13.1 FFIRN/FUD 1349/1adapted - Def MIP-NDAG PAREXT 1000009
Parachute, low altitude extraction To deplane from the aircraft while it is airborne with a parachute that will pull the cargo out of the aircraft. ADatP-3 F068 BL12, BL13.1 FFIRN/FUD 1349/1adapted - Def MIP-NDAG PARLAE 1000010
Parachute, manual To deplane by jumping out of the aircraft while it is airborne and manually deploy a parachute before reaching the ground. ADatP-3 F068 BL12, BL13.1 FFIRN/FUD 1349/1adapted - Def MIP-NDAG PARMAN 1000011
Paraglider To deplane by jumping out of the aircraft while it is airborne and deploy a paraglider before reaching the ground. ADatP-3 F068 BL12, BL13.1 FFIRN/FUD 1349/1adapted - Def MIP-NDAG PARGLD 1000012
Rope To deplane using a rope while the aircraft is airborne. APP-9 - Def ATCCIS ROPE 1000004
Wedge To deplane cargo from the aircraft by dropping it, using a wedge system. ADatP-3 BL12, BL13.1 FFIRN/FUD 1349/1adapted - Def MIP-NDAG WEDGE 1000013
Winch To deplane by using a winch-operated cable while the aircraft is airborne. APP-9 - Def ATCCIS WINCH 1000005
