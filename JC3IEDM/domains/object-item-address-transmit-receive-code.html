Name: object-item-address-transmit-receive-code
Definition: The specific value that denotes the OBJECT-ITEM usage of an ELECTRONIC-ADDRESS in terms of transmission and reception.
Definition Source: MIP-NDAG

Value Definition Source Physical Value Identifier
