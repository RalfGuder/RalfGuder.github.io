Domain
Name: runway-pavement-type-code
Definition: The specific value that represents the type of pavement classification, which is part of the standard ICAO (International Civil Aviation Organization) method of reporting pavement strength for pavements with bearing strength greater than 5,700 kilograms (12,500 pounds).
Definition Source: DAFIF (Adapted ICAO Publication: Annex 14)
Allowable Values:
Value Definition Source Physical Value Identifier
Flexible The pavement type has a flexible classification. DAFIF (adapted ICAO Publication) STANAG 7131 F 1000001
Rigid The pavement type has a rigid classification. DAFIF (adapted ICAO Publication) STANAG 7131 R 1000002
Usage:
Entity Attribute Opt
