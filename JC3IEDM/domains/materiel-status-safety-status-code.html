Domain
Name: materiel-status-safety-status-code
Definition: The specific value that represents the arming state of a specific MATERIEL that is a weapon or ammunition.
Definition Source: MIP-DMSWG - US JP 1-02
Allowable Values:
Value Definition Source Physical Value Identifier
Armed The specific MATERIEL is ready to fire or explode. AAP-6 - Def adapted ARMED 1000003
Neutralized The specific MATERIEL has been rendered incapable of firing or exploding, although it may remain dangerous to handle. AAP-6 adapted NUTRLD 1000004
Safe The state in which material cannot function and in which it is safe to handle and transport. AAP-6 - Def adapted SAFE 1000002
Unassembled The specific MATERIEL is dismantled or taken apart so that it is not operable, but it can be operable if reassembled. ATCCIS UNASMB 1000001
Not known It is not possible to determine which value is most applicable. ATCCIS NKN 1000005
Usage:
Entity Attribute Opt
