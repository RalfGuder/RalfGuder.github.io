Domain
Name: aircraft-type-airframe-design-code
Definition: The specific value that represents the design of the airframe of an AIRCRAFT-TYPE.
Definition Source: MIP-NDAG
Allowable Values:
Value Definition Source Physical Value Identifier
Autogyro An aircraft powered by a conventional propeller and a freewheeling, horizontal rotor. AIntP-3 - Def adapted OED AUTOGY 1000001
Balloon An unpowered lighter than air aircraft. MIP-NDAG BALOON 1000002
Bomber An aircraft capable of air operations using air to ground ordnance to destroy or neutralize enemy forces or installations. ADatP-3 BL10.0.1, BL13.1 FFIRN/FUD 1650/6 - Def OED adapted BOMBER 1000003
Dirigible A powered lighter than air aircraft. MIP-NDAG DIRIG 1000004
Fighter A generic term to describe a type of fast and manoeuvrable fixed wing aircraft capable of tactical air operations against air and/or surface targets. AAP-6 FIGHTR 1000005
Glider A fixed wing aircraft that flies without an engine. ADat-P3 BL12, BL13.1 FFIRN/FUD 1650/6 - Def OED adapted GLIDER 1000006
Helicopter A rotary wing aircraft capable of atmospheric flight and dependent on rotating blades for lift. MIP-NDAG HELO 1000007
Satellite A space vehicle that orbits the earth. MIP-NDAG SATEL 1000011
Transport An aircraft designed primarily for the carriage of personnel and/or cargo, and/or fuel. MIP-NDAG CARGO 1000008
Not known It is not possible to determine which value is most applicable. ATCCIS NKN 1000009
Not otherwise specified The appropriate value is not in the set of specified values. ATCCIS NOS 1000010
