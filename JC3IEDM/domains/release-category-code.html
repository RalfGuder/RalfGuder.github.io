Domain
Name: release-category-code
Definition: The specific value that represents the chemical, biological, or radiological class of release.
Definition Source: CBRN COI
Allowable Values:
Value Definition Source Physical Value Identifier
Continuous A continuous discharge of a contaminant release in a release other than attack (ROTA) event. CBRN COI CONT 1000001
Single release of a cloud A single discharge of a contaminant release in a release other than attack (ROTA) event. CBRN COI PUFF 1000002
Spraying The spraying of a contaminant in a release other than attack (ROTA) event. CBRN COI SPRAY 1000003
Not known It is not possible to determine which value is most applicable. ATCCIS NKN 1000004
Not otherwise specified The appropriate value is not in the set of specified values. ATCCIS NOS 1000005

Entity Attribute Opt
