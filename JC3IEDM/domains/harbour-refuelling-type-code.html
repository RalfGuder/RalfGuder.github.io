Domain
Name: harbour-refuelling-type-code
Definition: The specific value that represents the refuelling means available at the specific HARBOUR.
Definition Source: FIMU/MIP
Allowable Values:
Value Definition Source Physical Value Identifier
Bunkering, barge A mobile facility that is a barge whose sole purpose is the refuelling of vessels. MIP BKRBRG 1000001
Fixed installation A static facility from which vessels can refuel. MIP FXDINS 1000002
Tanker, road A mobile facility that is a road fuel tanker used for refuelling vessels whilst moored alongside a jetty, basin, quay or berth. MIP TNKRRD 1000003
Not otherwise specified The appropriate value is not in the set of specified values. ATCCIS NOS 1000004
