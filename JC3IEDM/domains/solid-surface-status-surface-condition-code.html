Domain
Name: solid-surface-status-surface-condition-code
Definition: The specific value that represents the physical status of a solid surface area.
Definition Source: MIP-NDAG
Allowable Values:
Value Definition Source Physical Value Identifier
Dust A characterisation of an area covered with fine dry particles of matter. ADatP-3 BL12, BL13.1 FFIRN/FUD 1783/1 - Def OED adapted DUST 1000001
Earth A characteristic of an area covered with soil. ADatP-3 BL12, BL13.1 FFIRN/FUD 1783/1 - Def OED adapted EARTH 1000010
Flood A characterisation of an area covered or submerged temporarily in water. ADatP-3 BL12 1783/1, BL13.1 FFIRN/FUD 1270/2 adapted - Def OED adapted FLOOD 1000011
Ice A characterisation of an area covered with a layer or mass of frozen water. ADatP-3 BL12, BL13.1 FFIRN/FUD 1783/1 - Def OED adapted ICE 1000004
Sand A characterisation of an area covered with small, loose grains of worn or disintegrated rock. ADatP-3 BL12, BL13.1 FFIRN/FUD 1783/1 - Def OED adapted SAND 1000006
Snow A characterisation of an area covered with snow. ADatP-3 BL12, BL13.1 FFIRN/FUD 1783/1 - Def OED adapted SNOW 1000008
Not known It is not possible to determine which value is most applicable. ATCCIS NKN 1000012
Not otherwise specified The appropriate value is not in the set of specified values. ATCCIS NOS 1000009
