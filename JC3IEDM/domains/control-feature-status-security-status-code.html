Domain
Name: control-feature-status-security-status-code
Definition: The specific value that represents the protection status of the site encompassed by a specific CONTROL-FEATURE.
Definition Source: ATCCIS
Allowable Values:
Value Definition Source Physical Value Identifier
Guarded The site associated to the specific control-feature is being guarded so as to control entry or exit. ATCCIS GUARDD 1000001
None No special action has been taken to guarantee the safety of the site associated to the specific control feature. ATCCIS NONE 1000002
Secured The site associated to the specific control-feature is protected or safe. ATCCIS SECURD 1000003
Not known It is not possible to determine which value is most applicable. ATCCIS NKN 1000004
Usage:
Entity Attribute Opt
CONTROL-FEATURE-STATUS control-feature-status-security-status-code OP
