Name: aircraft-type-design-role-code
Definition: The specific value that represents the designed role of an AIRCRAFT-TYPE.
Definition Source: MIP-NDAG
Allowable Values:
Value Definition Source Physical Value Identifier
Defensive The major role of this aircraft is defensive. MIP-NDAG DEF 1000001
Multi role An aircraft capable of carrying out more than one role. MIP-NDAG MULTI 1000002
Offensive The major role of this aircraft is offensive. MIP-NDAG OFF 1000003
Support The major role of this aircraft is support. MIP-NDAG SUPPRT 1000004
Not known It is not possible to determine which value is most applicable. ATCCIS NKN 1000005
Not otherwise specified The appropriate value is not in the set of specified values. ATCCIS NOS 1000006
