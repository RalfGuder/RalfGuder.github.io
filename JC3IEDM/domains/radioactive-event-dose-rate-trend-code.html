Domain
Name: radioactive-event-dose-rate-trend-code
Definition: The specific value that represents the rate of change of the ionising radiation emitted by a radioactive materiel.
Definition Source: CBRN COI
Allowable Values:
Value Definition Source Physical Value Identifier
Background A background radiation reading of any radioactive material with respect to time. ATP-45 BACK 1000010
Decreasing A decreasing radiation intensity of any radioactive material with respect to time. ATP-45 DECR 1000004
Increasing An increasing radiation intensity of any radioactive material with respect to time. ATP-45 INCR 1000005
Initial The initial radiation intensity of any radioactive material at the moment of the nuclear event. ATP-45 INIT 1000006
Peak A maximum radiation intensity of any radioactive material with respect to time. ATP-45 PEAK 1000007
Same A constant radiation intensity of any radioactive material with respect to time. ATP-45 SAME 1000008
Not known It is not possible to determine which value is most applicable. ATCCIS NKN 1000009

Entity Attribute Opt
