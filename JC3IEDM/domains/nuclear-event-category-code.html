Domain
Name: nuclear-event-category-code
Definition: The specific value that represents the class of NUCLEAR-EVENT.
Definition Source: CBRN COI
Allowable Values:
Value Definition Source Physical Value Identifier
NUCLEAR-WEAPON-EVENT A NUCLEAR-EVENT that involves the detonation of a nuclear device. CBRN COI NUCWEP 1000007
Nuclear attack The delivery of a nuclear weapon against a target. MIP-NDAG NUCATT 1000001
Nuclear release other than attack (ROTA) The release of nuclear material into the environment intentionally or accidentally but not for the intended purpose of conducting an attack. CBRN COI NROTA 1000002
Nuclear sampling The action of detecting a radiological contaminant. MIP-NDAG NUCSMP 1000003
Nuclear test The explosion of a nuclear device for test purpose. MIP-NDAG NUCTST 1000004
Nuclear war alert The state of readiness caused by the possibility of atomic bombings. MIP-NDAG NUCWAR 1000005
Nuclear waste disposal Removing nuclear polluted substances to a designated storage area. MIP-NDAG NUCWST 1000006
Not known It is not possible to determine which value is most applicable. ATCCIS NKN 1000008
