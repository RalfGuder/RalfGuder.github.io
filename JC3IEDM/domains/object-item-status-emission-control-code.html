Name: object-item-status-emission-control-code
Definition: The specific value that represents the emission control status of a specific OBJECT-ITEM.
Definition Source: ATCCIS
Allowable Values:
Value Definition Source Physical Value Identifier
EMCON 1 Electronic Silence is a deliberate prohibition of electronic radiation to prevent the emission of significant electromagnetic signals. QSTAG Number 1102 Edition 1 EMCON1 1000001
EMCON 2 Radio Silence is a restriction placed on the use of all or certain radio equipment. Any commander at any level may impose radio silence. QSTAG Number 1102 Edition 1 EMCON2 1000002
EMCON 3 No restrictions, normal operations. QSTAG Number 1102 Edition 1 EMCON3 1000003
