Name: network-capacity-bandwidth-code
Definition: The specific value that represents a bandwidth capacity that is supported by a NETWORK.
Definition Source: MIP
Allowable Values:
Value Definition Source Physical Value Identifier
1 Gbps A digital bandwidth value representing a transfer speed of 1 Gigabits per second. MIP 1GBPS 1000017
10 Gbps A digital bandwidth value representing a transfer speed of 10 Gigabits per second. MIP 10GBPS 1000018
10 Mbps A digital bandwidth value representing a transfer speed of 10 Megabits per second. MIP 10MBPS 1000015
100 Mbps A digital bandwidth value representing a transfer speed of 100 Megabits per second. MIP 100MBP 1000016
56 Kbps A digital bandwidth value representing a transfer speed of 56 Kilobits per second. MIP 56KBPS 1000013
64 Kbps A digital bandwidth value representing a transfer speed of 64 Kilobits per second. MIP 64KBPS 1000014
E1 A dedicated point-to-point transmission line of 2048Kbps bandwidth capacity. MIP E1 1000001
E2 A dedicated point-to-point transmission line of 8192 Kbps bandwidth capacity. MIP E2 1000002
E3 A dedicated point-to-point transmission line of 32768 Kbps bandwidth capacity. MIP E3 1000003
Eurocom A dedicated point-to-point transmission line of 512 Kbps bandwidth capacity. MIP EUROCM 1000004
FDDI A 100 Mbps token passed ring network. MIP FDDI 1000005
Frame relay A 1.544Mbps permanent virtual circuit that uses a packet switching scheme. MIP FRMRLY 1000006
ISDN A 144Kbps basic rate dial-up line. MIP ISDN 1000007
SONET A fibre optic NETWORK that delivers data at speeds up to 622 Mbps, and beyond. (Synchronous Optical NETwork). MIP SONET 1000008
Switched 56 A circuit-switched 56Kbps leased line. MIP SWTD56 1000009
T1 A dedicated point-to-point transmission line of 1.544Mbps bandwidth capacity. MIP T1 1000010
T3 A dedicated point-to-point transmission line of 45 Mbps bandwidth capacity. MIP T3 1000011
Not otherwise specified The appropriate value is not in the set of specified values. MIP NOS 1000012
