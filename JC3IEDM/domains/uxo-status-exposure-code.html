Domain
Name: uxo-status-exposure-code
Definition: The specific value that represents the visual status of a specific Unexploded Explosive Ordnance.
Definition Source: ATCCIS
Allowable Values:
Value Definition Source Physical Value Identifier
Fully exposed A status indicating that a specific Unexploded Explosive Ordnance is fully exposed. MIP-NDAG FULEXP 1000001
Partially exposed A status indicating that a specific Unexploded Explosive Ordnance is partially exposed. MIP-NDAG PRTEX 1000002
Partially exposed, body A status indicating that the body of a specific Unexploded Explosive Ordnance is partially exposed. MIP-NDAG PRTEXB 1000003
Partially exposed, nose A status indicating that the nose of a specific Unexploded Explosive Ordnance is partially exposed. MIP-NDAG PRTEXN 1000004
Partially exposed, side A status indicating that the side of a specific Unexploded Explosive Ordnance is partially exposed. MIP-NDAG PRTEXS 1000005
Partially exposed, tail A status indicating that the tail of a specific Unexploded Explosive Ordnance is partially exposed. MIP-NDAG PRTEXT 1000006
Unexposed A status indicating that a specific Unexploded Explosive Ordnance is not exposed. MIP-NDAG UNEXPD 1000007
Not known It is not possible to determine which value is most applicable. ATCCIS NKN 1000008
Not otherwise specified The appropriate value is not in the set of specified values. ATCCIS NOS 1000009
