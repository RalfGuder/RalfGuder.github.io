Domain
Name: approach-direction-category-code
Definition: The specific value that differentiates between left, right and centre parallel runways, Short Takeoff and Landing (STOL) or true as applicable.
Definition Source: DAFIF (Adapted ICAO Publication: Annex 14)
Allowable Values:
Value Definition Source Physical Value Identifier
Centre The specific value that specifies the centre runway approach for parallel runways. Adapted DAFIF C 1000001
Left The specific value that specifies the left side runway approach for parallel runways. Adapted DAFIF L 1000002
Right The specific value that specifies the right side runway approach for parallel runways. Adapted DAFIF R 1000003
STOL The specific value that specifies short take-off and landing approach direction. Adapted DAFIF S 1000004
True The specific value, which specifies that the approach direction is comparable to the true heading not magnetic heading. Adapted DAFIF T 1000005
Usage:
Entity Attribute Opt
APPROACH-DIRECTION approach-direction-category-code OP
