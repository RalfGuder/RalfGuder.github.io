Domain
Name: unit-type-supplementary-specialisation-code
Definition: The specific value that supplements the designation of a particular UNIT-TYPE.
Definition Source: ATCCIS
Allowable Values:
Value Definition Source Physical Value Identifier
Air A UNIT-TYPE that, whatever its primary function is, employs principally air assets. SHAPE DAO AIR 1000012
Air assault A UNIT-TYPE that, whatever its primary function is, employs helicopter assets as an integral element of its assault operations. APP-6A - Def MIP ARASLT 1000001
Airborne A UNIT-TYPE that, whatever its primary function is, is specially trained to carry out operations, either by paradrop or air landing, following an air movement. APP-6A - Def adapted AAP-6 (2002) AIRBRN 1000002
Amphibious A UNIT-TYPE that, whatever its primary function is, is able to conduct an operation launched from the sea by military forces against a hostile or potentially hostile shore. APP-6A - Def adapted AAP-6(V) AMPH 1000003
Arctic A UNIT-TYPE that, whatever its primary function is, is able to perform a mission in arctic environment. APP-6A - Def MIP ARCTIC 1000004
Armoured A UNIT-TYPE that, whatever its primary function is, employs principally armoured assets. APP-6A - Def ATCCIS ARMORD 1000005
Ground A UNIT-TYPE that, whatever its primary function is, employs principally ground assets. SHAPE DAO GROUND 1000013
Light unit A UNIT-TYPE that, whatever its primary function is, is equipped with materiel of small weight. APP-6A - Def MIP LIGHT 1000006
Marine A UNIT-TYPE that is capable of performing marine operations in addition to its primary function. APP-6A - Def MIP-NDAG MARINE 1000007
Mechanised A UNIT-TYPE that, whatever its primary function is, employs principally mechanised assets. APP-6A - Def ATCCIS MECH 1000008
Motorised A UNIT-TYPE that, whatever its primary function is, employs principally motorised assets. APP-6A - Def ATCCIS MTRSD 1000009
Mountain A UNIT-TYPE that, whatever its primary function is, is dedicated to conduct military operations in mountainous areas. APP-6A - Def adapted OED MOUNTN 1000010
Naval A UNIT-TYPE that, whatever its primary function is, is dedicated to conduct operations in a maritime environment. APP-6A - Def MIP NAVAL 1000011
