Name: bridge-type-design-type-code
Definition: The specific value that represents the design class of BRIDGE-TYPE.
Definition Source: ATCCIS
Allowable Values:
Value Definition Source Physical Value Identifier
Arch A bridge supported by curved structures. AEngrP-2(A) - Def COD ARCH 1000001
Box-girder A bridge using large iron or steel beams or compound structure for bearing loads. AEngrP-2(A) - Def COD BXGRDR 1000002
Cantilever A bridge made of cantilevers projecting from the piers and connected by girders. AEngrP-2(A) - Def COD CNTLVR 1000003
Ferry A movable bridge represented by a ferry moving back and forth between fixed bridging sites. ATCCIS FERRY 1000004
Floating, boat A bridge constructed using small boats. ADatP-3 BL11, BL13.1 FFIRN/FUD 1650/6 adapted - Def adapted COD FLTBOT 1000018
Floating, not otherwise specified A bridge on pontoons etc. AEngrP-2(A) - Def COD FLTNOS 1000017
Floating, pontoon A bridge constructed on pontoons. ADatP-3 BL11, BL13.1 FFIRN/FUD 1650/6 adapted - Def adapted COD FLTPNT 1000019
Lift A bridge that can be raised to allow the passage of ships. AEngrP-2(A) - Def adapted COD LIFT 1000006
Military, medium girder A bridge using medium iron or steel beams or compound structure for bearing loads. UK SOHB MILMDG 1000007
Military, vehicle launched A bridge that is carried on a vehicle and is deployed from the vehicle for immediate use. UK SOHB MILVHL 1000009
Raft A movable bridge represented by a raft moving back and forth between fixed bridging sites, usually along wires connected to both these sites. ATCCIS RAFT 1000010
Slab A bridge made by use of a flat broad fairly thick usu. square or rectangular piece of solid material, esp. stone. AEngrP-2(A) - Def adapted COD SLAB 1000011
Stringer A bridge made by use of a longitudinal structural member in a framework. AEngrP-2(A) - Def adapted COD STRNGR 1000012
Suspension A bridge with a roadway suspended from cables supported by structures at each end. AEngrP-2(A) - Def COD SSPNSN 1000013
Swing A bridge that can be swung to one side to allow the passage of ships. AEngrP-2(A) - Def COD SWING 1000014
Truss A bridge made by use of a framework, e.g. of rafters and struts. AEngrP-2(A) - Def adapted COD TRUSS 1000015
Not known It is not possible to determine which value is most applicable. ATCCIS NKN 1000020
Not otherwise specified The appropriate value is not in the set of specified values. ATCCIS NOS 1000016
