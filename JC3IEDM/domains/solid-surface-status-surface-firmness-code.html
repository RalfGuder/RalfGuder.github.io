Domain
Name: solid-surface-status-surface-firmness-code
Definition: The specific value that represents the firmness of a surface area, in terms of its ability to support land vehicles or helicopters.
Definition Source: MIP-NDAG
Allowable Values:
Value Definition Source Physical Value Identifier
Hard The surface area is able to support the helicopter and can be used by two-wheel drive vehicles or four-wheel drive vehicles and trailers. ADatP-3 BL12.2, BL13.1 FFIRN/FUD 1782/1 - Def BL13.1 FFIRN/FUD 1782/1 adapted HARD 1000004
Moderate The surface area can be used by three or four ton vehicles which should be able to start from rest using four-wheel drive. ADatP-3 BL12.2, BL13.1 FFIRN/FUD 1782/1 - Def BL13.1 FFIRN/FUD 1782/1 adapted MODER 1000002
Soft On the surface area, four-wheel drive vehicles cannot start from rest but might be able to cross if already on the move. ADatP-3 BL12.2, BL13.1 FFIRN/FUD 1782/1 - Def BL13.1 FFIRN/FUD 1782/1 adapted SOFT 1000003
Very soft Wheeled vehicles cannot progress across the surface. ADatP-3 BL12.2, BL13.1 FFIRN/FUD 1782/1 - Def BL13.1 FFIRN/FUD 1782/1 adapted VSOFT 1000005
Usage:
Entity Attribute Opt
