Domain
Name: light-moon-phase-code
Definition: The specific value that represents the phase of the moon for a specific LIGHT.
Definition Source: ATCCIS
Allowable Values:
Value Definition Source Physical Value Identifier
Full moon The moon with its entire disc illuminated. OED FUL 1000002
New moon The moon when first seen as a slender crescent shortly after its conjunction with the sun. OED NEW 1000004
Waning moon The moon is decreasing in phase. OED WAN 1000003
Waxing moon The moon is increasing in phase. OED WAX 1000001
Usage:
Entity Attribute Opt
LIGHT light-moon-phase-code OP
