Name: ied-tactical-characterization-emplacement-code
Definition: The specific value that describes where the device is positioned.
Definition Source: WTI IED Lexicon DEC 2008 adapted
Allowable Values:
Value Definition Source Physical Value Identifier
In/on IED emplaced inside or directly on an object. WTI IED Lexicon DEC 2008 adapted - Def WTI IED Lexicon DEC 2008 adapted INON 1000001
Overhead IED emplaced in a structure above the intended target. WTI IED Lexicon DEC 2008 adapted - Def WTI IED Lexicon DEC 2008 adapted OVERHD 1000002
Subsurface IED emplaced under the surface or below the intended target, i.e. buried, in a culvert, underwater. WTI IED Lexicon DEC 2008 adapted - Def WTI Handbook SEP 2009 SUBSFC 1000003
Surface IED emplaced directly on the ground or water surface. WTI IED Lexicon DEC 2008 adapted - Def WTI Handbook SEP 2009 SURFAC 1000004
Not known It is not possible to determine which value is most applicable. ATCCIS NKN 1000006
Not otherwise specified The appropriate value is not in the set of specified values. ATCCIS NOS 1000005
