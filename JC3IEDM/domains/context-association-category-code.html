Domain
Name: context-association-category-code
Definition: The specific value that represents the type of relationship between the subject CONTEXT and the object CONTEXT in a specific CONTEXT-ASSOCIATION.
Definition Source: MIP-NDAG
Allowable Values:
Value Definition Source Physical Value Identifier
Is next after The subject CONTEXT follows the object CONTEXT sequentially without negating it. The value is to be used to establish a time-ordered sequence of CONTEXTs. MIP-NDAG ISNEXT 1000001
Is part of/Is sub-context of The subject CONTEXT is included in the object CONTEXT. MIP-NDAG ISPART 1000002
Supersedes The subject CONTEXT negates the object CONTEXT by replacing it. MIP-NDAG SPRCDS 1000003
Supplements The subject CONTEXT provides amplifying information with respect to the object CONTEXT. MIP-NDAG SPPLMN 1000004
Usage:
Entity Attribute Opt
