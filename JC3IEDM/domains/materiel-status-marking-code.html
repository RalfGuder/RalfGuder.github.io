Domain
Name: materiel-status-marking-code
Definition: The specific value that represents the type of marking of a specific MATERIEL.
Definition Source: MIP-NDAG
Allowable Values:
Value Definition Source Physical Value Identifier
Numbers A set of integers. Adapted OED NUMBER 1000001
Stripe A long narrow band distinguished, as by colour, from the surrounding material or surface. This value includes band. Adapted OED STRIPE 1000002
Stripes Multiple long narrow bands distinguished, as by colour, from the surrounding material or surface. This value includes bands. Adapted OED STRIPS 1000003
Symbols A token for identification. Adapted OED SYMBOL 1000004
Writing Language symbols or characters written or imprinted on a surface. Adapted OED WRITNG 1000005
Not known It is not possible to determine which value is most applicable. ATCCIS NKN 1000006
Not otherwise specified The appropriate value is not in the set of specified values. ATCCIS NOS 1000007
Usage:
Entity Attribute Opt
