Domain
Name: fire-capability-descriptor-code
Definition: The specific value that represents the FIRE-CAPABILITY that is being quantified.
Definition Source: MIP-NDAG
Allowable Values:
Value Definition Source Physical Value Identifier
Burst fire rate The numeric value that denotes the rate at which projectiles can be fired in a short but rapid burst. MIP-NDAG BRTFIR 1000001
Maximum fire rate The numeric value that denotes the highest rate at which projectiles can be fired for a limited period of time. JCS Pub 1-02 MRATFR 1000002
Maximum range The longest distance that can be achieved. MIP-NDAG MRANGE 1000003
Minimum range The shortest distance that can be achieved. MIP-NDAG NRANGE 1000004
Safety distance factor The numeric value that denotes the minimum distance to be maintained between own troops and the intended target. ACT SFTDST 1000005
Sustained fire rate The numeric value that denotes the rate at which projectiles can be fired repeatedly for a prolonged period of time. MIP-NDAG SSTRTF 1000006
Usage:
Entity Attribute Opt
FIRE-CAPABILITY fire-capability-descriptor-code OP
