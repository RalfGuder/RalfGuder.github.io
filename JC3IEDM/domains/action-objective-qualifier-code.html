Domain
Name: action-objective-qualifier-code
Definition: The specific value that represents a restriction or other qualification applicable to a specific ACTION-OBJECTIVE for a specific ACTION.
Definition Source: ATCCIS
Allowable Values:
Value Definition Source Physical Value Identifier
Authorised The ACTION-OBJECTIVE is authorised without restriction. ATCCIS AUTH 1000001
Do not attack The specific ACTION-OBJECTIVE (e.g. hospitals or friendly forces) must not be attacked. MIP DONTAT 1000012
No exploitation east of line The specific ACTION-OBJECTIVE is authorised with the restriction that there is to be no exploitation east of it. MIP NEEL 1000002
No exploitation north of line The specific ACTION-OBJECTIVE is authorised with the restriction that there is to be no exploitation north of it. MIP NENL 1000003
No exploitation south of line The specific ACTION-OBJECTIVE is authorised with the restriction that there is to be no exploitation south of it. MIP NESL 1000004
No exploitation west of line The specific ACTION-OBJECTIVE is authorised with the restriction that there is to be no exploitation west of it. MIP NEWL 1000005
Not authorised The specific ACTION-OBJECTIVE is not authorised. ATCCIS NOTA 1000007
Stay above The specific ACTION-OBJECTIVE is authorised with the restriction that there is to be no movement below or within it. MIP-NDAG STAYAB 1000008
Stay below The specific ACTION-OBJECTIVE is authorised with the restriction that there is to be no movement above or within it. MIP-NDAG STAYBL 1000009
Stay inside The specific ACTION-OBJECTIVE is authorised with the restriction that there is to be no movement outside it. MIP STAYIN 1000010
Stay outside The specific ACTION-OBJECTIVE is authorised with the restriction that there is to be no movement inside it. MIP STAYOT 1000011
