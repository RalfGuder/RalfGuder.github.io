Domain
Name: ied-tactical-characterization-use-sequence-code
Definition: The specific value that identifies enemy use of an IED as a primary, secondary, or tertiary form of attack, independent of intended outcome.
Definition Source: WTI IED Lexicon DEC 2008 adapted
Allowable Values:
Value Definition Source Physical Value Identifier
Primary This is the initial IED employed in the target area. WTI IED Lexicon DEC 2008 adapted - Def WTI IED Lexicon DEC 2008 adapted PRI 1000001
Secondary This is an additional device emplaced in the target area to attack individuals or vehicles after the initial event. WTI IED Lexicon DEC 2008 adapted - Def WTI Handbook SEP 2009 SEC 1000002
Tertiary This is an additional device emplaced in the target area to attack individuals or vehicles after the initial and secondary events. WTI IED Lexicon DEC 2008 adapted - Def WTI Handbook SEP 2009 TER 1000003
Not known It is not possible to determine which value is most applicable. ATCCIS NKN 1000004

Entity Attribute Opt
