Name: object-item-alias-category-code
Definition: The specific value that represents the class of OBJECT-ITEM-ALIAS.
Definition Source: MIP-NDAG
Allowable Values:
Value Definition Source Physical Value Identifier
Alternate name A designation of the common additional reference given to an OBJECT-ITEM. MIP-NDAG ALTNAM 1000002
Elint notation A 5-character ELINT notation for non-communications electronics emissions related to such equipments. ADatP-3 BL13.1 FFIRN/FUD 1012/34 - Def BL13.1 FFIRN/FUD 1012/34 adapted ELINT 1000004
Emissions sorting code A 4-character code which facilitates reporting equipment type that has no associated ELINT notation. ADatP-3 BL13.1 FFIRN/FUD 1012/34 - Def BL13.1 FFIRN/FUD 1012/34 EMSSNS 1000005
Geolocation A standard coded representation of a geolocation of a military organisation or a place with military significance. ADatP-3 BL12, BL13.1 FFIRN/FUD 1125/11 - Def BL13.1 FFIRN/FUD 1125/11 GEOLOC 1000001
Track identifier An identifier that is used in track reporting. ADatP-3 BL11, BL13.1 FFIRN/FUD 1012/60 - Def MIP-DMSWG TRACK 1000003
Unit designator The designator of the unit that is tasked to perform the mission, for example: "UNIT:ARS BROCKZETEL". ADatP-3 BL12.2, BL13.1 FFIRN/FUD 1028/43 - Def MIP-DMSWG UNITDS 1000006
