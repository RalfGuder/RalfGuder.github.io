Domain
Name: railway-signal-system-code
Definition: The specific value that represents the type of signal system used for the RAILWAY.
Definition Source: Adapted AIntP-3 (SIGNAL SYSTEM)
Allowable Values:
Value Definition Source Physical Value Identifier
Colour light The specific value that represents a signal, which gives indication by the colour of the light only. Adapted AIntP-3 CL 1000001
Colour position The specific value that represents a signal giving indications by the colour and position of two or more lights. Adapted AIntP-3 CP 1000002
Electric light The specific value that represents an electric light system. Adapted AIntP-3 EL 1000003
Inconclusive analysis The specific value that indicates that the analysis of the system is inconclusive. Adapted AIntP-3 Z 1000004
Position light The specific value that represents a signal to which indications are given by the position of two or more lights. Adapted AIntP-3 PL 1000005
Semaphore The specific value that represents a signal in which the day indicators are given by changing positions of a semaphore arm and the night indications are given by coloured lights. Adapted AIntP-3 SF 1000006
Not known It is not possible to determine which value is most applicable. ATCCIS NKN 1000007
