Domain
Name: reference-transmittal-type-code
Definition: The specific value that represents the means by which the artefact cited in a specific REFERENCE is transmitted to the recipient.
Definition Source: ATCCIS
Allowable Values:
Value Definition Source Physical Value Identifier
Courier message A message received by courier. ATCCIS COUMSG 1000001
E-mail message A message received through an E-mail system. ATCCIS EMLMSG 1000002
Electronic transfer The information was received through electronic transfer. MIP-NDAG ELCTRF 1000010
Fax message A message received by fax. ATCCIS FAXMSG 1000003
Phone message A message received by telephone. ATCCIS PHNMSG 1000004
Radio message A message received from a radio transmission. ATCCIS RADMSG 1000005
Secure fax message A message received by secure fax. ATCCIS SFXMSG 1000006
Telex message A message received by a certain kind of civilian means. ATCCIS TELEX 1000007
Video The information was received via video. MIP-NDAG VIDEO 1000011
Not known It is not possible to determine which value is most applicable. ATCCIS NKN 1000008
Not otherwise specified The appropriate value is not in the set of specified values. ATCCIS NOS 1000009
