Name: surveillance-capability-category-code
Definition: The specific value that represents the class of SURVEILLANCE-CAPABILITY.
Definition Source: ATCCIS
Allowable Values:
Value Definition Source Physical Value Identifier
Communication The ability to gain information by exploiting communications systems. Adapted JCS Pub 1-02 COM 1000005
Electronic The ability to gain information by exploiting non-communications electronic systems (generally pulsed). Adapted JCS Pub 1-02 ELC 1000006
Human The ability to gain information by exploiting human senses. Adapted JCS Pub 1-02 HUM 1000002
Imaging The ability to gain information by exploiting imaging systems. Adapted JCS Pub 1-02 IMG 1000003
Signal The ability to gain information from exploiting EMS (not imagery) systems. Adapted JCS Pub 1-02 SIG 1000001
Not known It is not possible to determine which value is most applicable. ATCCIS NKN 1000004
Not otherwise specified The appropriate value is not in the set of specified values. ATCCIS NOS 1000007
