Domain
Name: demolition-status-code
Definition: The specific value that represents the status of an object destined for demolition.
Definition Source: ATCCIS
Allowable Values:
Value Definition Source Physical Value Identifier
Abandoned incomplete The object of the demolition was abandoned before the destruction was executed. Demolition devices may still be present. ATCCIS ABNDIN 1000001
Cancelled The demolition of the object was cancelled either before demolition devices were attached or these devices were removed. ATCCIS CANCLD 1000002
Executed The object is demolished. ATCCIS EXECTD 1000003
Planned preliminary The object of the demolition is planned as a preliminary demolition; execution can proceed without reference to an authorised commander. ATCCIS PLNPRL 1000004
Planned reserve The object of the demolition is planned as a reserve demolition; execution is to be ordered by a specific authorised commander. ATCCIS PLNRES 1000005
Prepared for execution The object is prepared for demolition. ATCCIS PRPEXE 1000006
State 1 The object is at demolition state 1 (safe). The demolition devices are installed but not armed. ATCCIS STATE1 1000007
State 2 The object is at demolition state 2 (armed). The demolition devices are installed and armed; therefore the object is ready for immediate demolition. ATCCIS STATE2 1000008
Not known It is not possible to determine which value is most applicable. ATCCIS NKN 1000009
