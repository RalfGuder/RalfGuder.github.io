Domain
Name: mine-presence-code
Definition: The specific value that indicates whether a specific FACILITY or GEOGRAPHIC-FEATURE contains mines.
Definition Source: ATCCIS
Allowable Values:
Value Definition Source Physical Value Identifier
No There are no mines in the specific FACILITY or GEOGRAPHIC-FEATURE. Adapted APP-9 NO 1000001
Yes Mines are present in the specific FACILITY or GEOGRAPHIC-FEATURE. Adapted APP-9 YES 1000002
Not known It is not possible to determine whether mines are present. Adapted APP-9 NKN 1000003
Usage:
Entity Attribute Opt
FACILITY-STATUS facility-status-mine-presence-code OP
GEOGRAPHIC-FEATURE-STATUS geographic-feature-status-mine-presence-code OP
