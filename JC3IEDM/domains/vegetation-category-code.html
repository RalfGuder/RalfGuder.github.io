Domain
Name: vegetation-category-code
Definition: The specific value that represents the general type of vegetation.
Definition Source: MIP-NDAG
Allowable Values:
Value Definition Source Physical Value Identifier
Bare Land devoid of vegetation. ACE Conceptual Data Model BARE 1000001
Jungle An area where heat is seldom less than 80°F/27°C, humidity normally 80-95% and rainfall on average at least 80 inches per year and rugged terrain. AFM(UK) Vol 2 Part 2 JUNGLE 1000004
Plant cultivation An area containing crops or gardens. COD PLANT 1000012
Rangeland An area containing uncultured plants of low height. STANAG 7074 - Def MIP-NDAG RNGLND 1000013
Wetlands A swamp, marsh, or other usually saturated area containing vegetation adapted to wet conditions. STANAG 7074 - Def adapted COD WETLND 1000014
Woodland An area containing vegetation composed of tree species which have reached sufficient size for use as timber. STANAG 7074 - Def adapted FM 5-33 Terrain analysis, 11 JULY 1990 WODLND 1000015
Not known It is not possible to determine which value is most applicable. ATCCIS NKN 1000016
Not otherwise specified The appropriate value is not in the set of specified values. ATCCIS NOS 1000011
