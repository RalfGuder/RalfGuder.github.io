Domain
Name: consumable-materiel-type-issuing-element-code
Definition: The specific value that represents a standard unit in which a specific CONSUMABLE-MATERIEL-TYPE is made available.
Definition Source: ATCCIS
Allowable Values:
Value Definition Source Physical Value Identifier
Bale A large bound, often wrapped package of materiel. APP-9 - Def adapted OED BALE 1000008
Barrel A large cylindrical container having a flat top and bottom of equal diameter. APP-9 - Def adapted OED BARREL 1000009
Box A rectangular container usually having a lid or cover. APP-9 - Def adapted OED BOX 1000010
Bulk Issued en masse (usually applies to liquids). ATCCIS BLK 1000001
Case A container. APP-9 - Def adapted OED CASE 1000011
Coil A series of connected spirals or concentric rings. APP-9 - Def adapted OED COIL 1000012
Container A receptacle for holding or carrying material. APP-9 - Def adapted OED CONTNR 1000013
Crate A container, as a slatted wooden case. APP-9 - Def adapted OED CRATE 1000014
Day of supply A Day of supply (at combat rate) is the amount of consumable materiel required to enable a formation (unit i.e. Division, Brigade etc) to carry out operations for 1 day. Adapted UK JSP 101 DAYSPL 1000018
Drum Issued in drums, the size of which is recorded in the issuing-unit-of-measure-code (usually applies to liquids) (e.g., 200-litre drums). ATCCIS DRM 1000002
Jerrican A container that holds 20 litres. APP-9 - Def Manual of Army Ordnance services (UK Army) JERCAN 1000015
Pack Issuing unit of measure is packs, the size of which is recorded in the issuing-unit-of-measure-code and issuing-quantity. ATCCIS PAK 1000005
Pallet Issued in units of standard NATO pallets. ATCCIS PAL 1000006
Ration A fixed portion. APP-9 - Def adapted OED RATION 1000016
Round A single item of ammunition. APP-9 - Def adapted OED ROUND 1000017
Unit Issued singly. ATCCIS UNT 1000007
