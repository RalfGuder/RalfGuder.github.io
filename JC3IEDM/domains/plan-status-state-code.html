Name: plan-status-state-code
Definition: The specific value assigned to represent the condition for a specific PLAN.
Definition Source: MIP-NDAG
Allowable Values:
Value Definition Source Physical Value Identifier
Approved A state of a specific PLAN that allows sub-units to initiate their own planning processes. MIP OWG APPRVD 1000001
Cancelled A state of a specific PLAN to indicate that the PLAN will not be used or further developed. MIP OWG CANCEL 1000002
Withdrawn A state of a specific PLAN that permits the PLAN to be edited. MIP OWG WTHDRN 1000003
