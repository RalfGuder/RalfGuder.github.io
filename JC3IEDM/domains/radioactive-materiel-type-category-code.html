Name: radioactive-materiel-type-category-code
Definition: The specific value that represents the class of RADIOACTIVE-MATERIEL-TYPE.
Definition Source: AEP/ATP-45(B)
Allowable Values:
Value Definition Source Physical Value Identifier
Cesium-137 A radioactive isotope of the chemical element of atomic number 55, a soft, silvery, rare, extremely reactive metal of the alkali metal group. COD CESIUM 1000001
Cobalt-60 A radioactive isotope of the chemical element of atomic number 27, a hard silvery-white magnetic metal. COD COBALT 1000002
Fresh reactor fuel Nuclear reactor fuel that has not yet been used in a live nuclear reactor. ATCCIS FRFUEL 1000003
Iodine-133 A radioactive isotope of the chemical element of atomic number 53, a halogen forming black crystals and a violet vapour. COD IODINE 1000004
Nuclear release other than attack (ROTA) An intended or non-intended release of nuclear radiation from a non-militarily significant source or weapon. ATCCIS NROTA 1000006
Nuclear weapon fallout Radioactive residues originating in a nuclear explosion. ATCCIS NUCFLT 1000005
Plutonium-239 A radioactive isotope of the chemical element of atomic number 94, a dense silvery radioactive metal of the actinide series, used as a fuel in nuclear reactors and as an explosive in nuclear fission weapons. COD PLTNUM 1000007
Spent reactor fuel Nuclear reactor fuel that has been used in a live nuclear reactor. ATCCIS SPFUEL 1000008
Toxic industrial material A generic term for radioactive compounds in solid, liquid, aerosolised or gaseous form. These may be used, or stored for use, for industrial, commercial, medical, military or domestic purposes. Adapted ATP-45(B) TOXMAT 1000009
Not known It is not possible to determine which value is most applicable. ATCCIS NKN 1000010
Not otherwise specified The appropriate value is not in the set of specified values. ATCCIS NOS 1000011
