Domain
Name: reporting-data-timing-category-code
Definition: The specific value that represents the absolute, uncertain or relative timing for a specific REPORTING-DATA.
Definition Source: ATCCIS
Allowable Values:
Value Definition Source Physical Value Identifier
REPORTING-DATA-ABSOLUTE-TIMING A REPORTING-DATA that specifies effective date and time that are referenced to Universal Time. ATCCIS RDABST 1000001
REPORTING-DATA-RELATIVE-TIMING A REPORTING-DATA that specifies effective timing that is referenced to a specific ACTION-TASK. ATCCIS RDRELT 1000002
Timing not available A REPORTING-DATA that indicates that the start or end timing with respect to past, current, or future periods is not available. MIP TIMNA 1000003
Usage:
Entity Attribute Opt
REPORTING-DATA reporting-data-timing-category-code OP
