Domain
Name: minefield-land-persistence-code
Definition: The specific value that represents the option for terminating the effectiveness of a specific MINEFIELD-LAND.
Definition Source: ATCCIS
Allowable Values:
Value Definition Source Physical Value Identifier
Permanent A minefield, already existing and effective, until somebody clears it. ADatP-3 - Def ATCCIS AAP-19(D) Nov 2003 adapted - Def AAP-19(D) Nov 2003 adapted PERMAN 1000003
Remote activated destruction A minefield that can be destroyed by remote control. ADatP-3 AAP-19(D) Nov 2003 adapted REMOTE 1000004
Timed automatic destruction A minefield that is in place for a certain period of time after which it destroys itself. AAP-19(D) Nov 2003 adapted - Def AAP-19(D) Nov 2003 adapted SLFDST 1000005
Not known It is not possible to determine which value is most applicable. ATCCIS NKN 1000001
