Domain
Name: materiel-type-supply-class-code
Definition: The specific value that represents the NATO supply class of MATERIEL-TYPE.
Definition Source: ATCCIS
Allowable Values:
Value Definition Source Physical Value Identifier
Class I Items that are consumed by personnel and animals at an approximately uniform rate, irrespective of local changes in combat or terrain conditions, e.g. food and forage. STANAG 2961 CLS1 1000001
Class II Supplies for which allowances are established by tables of organisation and equipment, e.g. clothing, weapons, tools, spare parts, vehicles. STANAG 2961 CLS2 1000002
Class III Fuel and lubricants for all purposes, except for operating aircraft or for use in weapons such as flame-throwers, e.g. gasoline, fuel oil, greases, coal and coke. STANAG 2961 CLS3 1000003
Class IIIa Aviation fuel and lubricants. STANAG 2961 CLS3A 1000004
Class IV Supplies for which initial use allowances are not prescribed by approved issue tables. Normally includes fortification and construction materials, as well as additional quantities of items identical to those authorised for initial issue (Class II) such as additional vehicles. STANAG 2961 CLS4 1000005
Class V Ammunition, explosives and chemical agents of all types. STANAG 2961 CLS5 1000006
