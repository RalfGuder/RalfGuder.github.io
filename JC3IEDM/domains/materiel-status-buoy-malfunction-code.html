Domain
Name: materiel-status-buoy-malfunction-code
Definition: The specific value that represents the type of malfunction of a buoy.
Definition Source: ADatP-3 BL13.1 FFIRN/FUD 1065/1 adapted
Allowable Values:
Value Definition Source Physical Value Identifier
Buoy not in sight The buoy is no longer visible in the location/area deployed. ADatP-3 BL12.2, BL13.1 FFIRN/FUD 1065/1 - Def MIP-NDAG BUOYNT 1000003
No flag The buoy does not display a marker flag. ADatP-3 BL12.2, BL13.1 FFIRN/FUD 1065/1 - Def MIP-NDAG NOFLAG 1000001
No light The buoy no longer has an operable lighting mechanism. ADatP-3 BL12.2, BL13.1 FFIRN/FUD 1065/1 - Def MIP-NDAG NOLGHT 1000005
No radar reflection The buoy does not produce an acceptable radar reflection. ADatP-3 BL12.2, BL13.1 FFIRN/FUD 1065/1 - Def MIP-NDAG NORADR 1000002
Serviceable buoy has dragged its position The buoy has moved from its original placed location. ADatP-3 BL12.2, BL13.1 FFIRN/FUD 1065/1 - Def MIP-NDAG SERVCE 1000004
Usage:
Entity Attribute Opt
MATERIEL-STATUS materiel-status-buoy-malfunction-code OP
