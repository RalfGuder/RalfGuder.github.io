Domain
Name: network-service-category-code
Definition: The specific value that represents a general type of service that a specific NETWORK is intended to provide.
Definition Source: ATCCIS
Allowable Values:
Value Definition Source Physical Value Identifier
Data transfer A service for the electronic transfer of data. ATCCIS DATTRF 1000014
Facsimile A service provided by means of electronic scanning of a document, transmitting it as data by telecommunications links, and producing an exact copy at the receiver. Adapted OED FAX 1000003
Identification friend or foe A system using electromagnetic transmissions to which equipment carried by friendly forces automatically responds, for example, by emitting pulses, thereby distinguishing themselves from enemy forces. MIP - Def APP-6A IFF 1000015
Image A service for electronic transfer of static or slowly refreshed images. NDAG IMAGE 1000016
MCI MIP Common Interface service. MIP MCI 1000018
Message handling service An application service that provides a generalised facility for exchanging electronic messages between systems. ADatP-2 MHS 1000019
Tactical data link One or more data links organised within a given architecture and supporting a standard set of binary messages exchanged between users of that system, for use in military operations. ADatP-3 BL13.1 FFIRN/FUD 1576/1 adapted - Def MIP-DMSWG TDL 1000020
Video service A service for electronic transfer of video information. ATCCIS VIDSVC 1000010
Voice service A service for electronic transfer of voice information. ATCCIS VOCSVC 1000011
Not otherwise specified The appropriate value is not in the set of specified values. ATCCIS NOS 1000012

Entity Attribute Opt
