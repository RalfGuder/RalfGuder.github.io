Name: geographic-feature-terrain-code
Definition: The specific value that represents a tract of land.
Definition Source: MIP-NDAG
Allowable Values:
Value Definition Source Physical Value Identifier
Flat The terrain of the GEOGRAPHIC-FEATURE is characterised as broadly level. AIntP-3 - Def adapted OED FLAT 1000001
Hilly The terrain of the GEOGRAPHIC-FEATURE is characterised as having multiple hills. NATO HILLY 1000002
Mountainous The terrain of the GEOGRAPHIC-FEATURE is characterised as having many large natural elevations of the earth's surface rising abruptly from the surrounding level. ACE Conceptual Data Model MOUNTS 1000003
Undulating The terrain of the GEOGRAPHIC-FEATURE is characterised as rolling or wavy. ACE Conceptual Data Model UNDULT 1000004
Urban A topographical complex where man-made construction or high population density is the dominant feature. ATP-45(B) - Def FM 3-06 (US Army, June 2003, Urban Operations) URBAN 1000006

Not otherwise specified The appropriate value is not in the set of specified values. ATCCIS NOS 1000005
