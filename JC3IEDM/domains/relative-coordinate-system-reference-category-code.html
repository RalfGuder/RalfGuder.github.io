Domain
Name: relative-coordinate-system-reference-category-code
Definition: The specific value that represents the source of the reference for defining the origin and axial directions for a specific RELATIVE-COORDINATE-SYSTEM.
Definition Source: MIP-NDAG
Allowable Values:
Value Definition Source Physical Value Identifier
OBJECT-REFERENCE A RELATIVE-COORDINATE-SYSTEM that has its frame of reference defined by using the position and orientation of a specific OBJECT-ITEM at a given point in time. MIP-NDAG OBJREF 1000001
POINT-REFERENCE A RELATIVE-COORDINATE-SYSTEM that uses three specific POINTs to establish its frame of reference. MIP-NDAG PNTREF 1000002
Usage:
Entity Attribute Opt
