Domain
Name: action-event-detail-intended-outcome-code
Definition: The specific value that describes the immediate and direct goals or objectives of an enemy's attack.
Definition Source: WTI IED Lexicon DEC 2008 adapted
Allowable Values:
Value Definition Source Physical Value Identifier
Anti-air vehicle The intended result is to damage or destroy aircraft and/or their payload as well as to kill or wound individuals inside the aircraft. WTI IED Lexicon DEC 2008 adapted - Def WTI IED Lexicon DEC 2008 adapted AIRDEF 1000001
Anti-armour The intended result is to damage or destroy armoured vehicles and/or to kill or wound individuals inside armoured vehicles. WTI IED Lexicon DEC 2008 adapted - Def WTI IED Lexicon DEC 2008 adapted ARMDEF 1000002
Anti-infrastructure The intended result is to damage or destroy physical infrastructure. WTI IED Lexicon DEC 2008 adapted - Def WTI IED Lexicon DEC 2008 adapted INFDEF 1000003
Anti-personnel The intended result is to kill or wound people. WTI IED Lexicon DEC 2008 adapted - Def WTI IED Lexicon DEC 2008 adapted PERDEF 1000004
Anti-vehicle The intended result is to damage or destroy vehicles, excluding armoured vehicles, and/or their cargo as well as to kill or wound individuals inside such vehicles. WTI IED Lexicon DEC 2008 adapted - Def WTI IED Lexicon DEC 2008 adapted VEHDEF 1000005
Anti-vessel The intended result is to damage or destroy vessels. MIP OWG - Def MIP-DMSWG JAN 2010 VESDEF 1000006
TTP identification The intended result is to cause a reaction by forces in an effort to learn and understand employed tactics. This knowledge is then used by the attacker to plan new attacks incorporating the lessons learned to inflict additional casualties or to avoid countermeasures. (TTP - Tactics, Techniques and Procedures). WTI IED Lexicon DEC 2008 adapted - Def WTI IED Lexicon DEC 2008 adapted TTPID 1000007
Not known It is not possible to determine which value is most applicable. ATCCIS NKN 1000009
Not otherwise specified The appropriate value is not in the set of specified values. ATCCIS NOS 1000008

Entity Attribute Opt
