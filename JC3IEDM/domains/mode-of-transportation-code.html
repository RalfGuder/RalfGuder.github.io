Domain
Name: mode-of-transportation-code
Definition: The specific value that indicates the mode of transportation of a specific ROUTE-SEGMENT.
Definition Source: Adapted ADatP-3 FFIRN 1269
Allowable Values:
Value Definition Source Physical Value Identifier
Air Represents the general mode of transportation used in movement by air. ADatP-3 BL13.1 FFIRN/FUD 1269/5 - Def MIP-DMSWG AIR 1000001
Inland waterway Represents the general mode of transportation used in movement by inland waterways. ADatP-3 BL13.1 FFIRN/FUD 1269/5 - Def MIP-DMSWG IWT 1000002
Multi mode Represents transportation utilizing more than one mode of transportation. MIP-NDAG MULTI 1000003
Pipeline Represents the mode of transportation of materiel by pipeline. ADatP-3 BL13.1 FFIRN/FUD 1269/1, AAP-6 APR 2008 - Def MIP-DMSWG PIPE 1000004
Rail Represents the general mode of transportation used in surface movement by rail. ADatP-3 BL13.1 FFIRN/FUD 1269/5 - Def MIP-DMSWG RAIL 1000005
Road Represents the general mode of transportation used in surface movement by road. ADatP-3 BL13.1 FFIRN/FUD 1269/5 - Def MIP-DMSWG ROAD 1000006
Sea Represents the general mode of transportation used in movement by sea. ADatP-3 BL13.1 FFIRN/FUD 1269/5 - Def MIP-DMSWG SEA 1000007
Terrain Represents the general mode of transportation used in surface movement other than road and rail. MIP-NDAG TERR 1000008
Usage:
Entity Attribute Opt
ROUTE route-mode-of-transportation-code OP
ROUTE-SEGMENT route-segment-mode-of-transportation-code OP
