Value Definition Source Physical Value Identifier

Sand A characterisation of an area that is composed mainly of small, loose grains of worn or disintegrated rock. ADatP-3 BL12, BL13.1 FFIRN/FUD 1783/1 - Def MIP-DMSWG SAND 1000006
Snow A characterisation of an area which is composed of snow. ADatP-3 BL12, BL13.1 FFIRN/FUD 1783/1 - Def MIP-DMSWG SNOW 1000008
