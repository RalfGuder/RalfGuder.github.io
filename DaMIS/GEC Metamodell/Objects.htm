A

Begriff

Definition

ACCREDITATION-CONTEXT

The official certification that a model, simulation, or federation of models and simulations and its associated data are acceptable for use for a specific purpose.

ACCREDITATION-ELEMENT

Evaluation data for a certain REFERENCE-ELEMENT, which result from an official certification for use for a specific purpose.

AGGREGATION-ELEMENT

The grouping of entities while preserving the effects of entity behavior and interaction while grouped.

APPLICATION-DATA-ELEMENT

A DATA-REFERENCE-ELEMENT, which denotes application data referred as ISO IRDS Framework Layer 1 data. An example is a certain parameter, which describes a specific simulation result.

C

Begriff

Definition

CONTEXT-ELEMENT

A reference to one or more REFERENCE-ELEMENTs. Contexts refer multiple REFERENCE-ELEMENTs to indicate a logical association to, e.g. a simulation-based project or a certain data "snapshot" or event.

CONTEXT-ELEMENT-ASSOCIATION

A association between a subject CONTEXT-ELEMENT and a specific object CONTEXT-ELEMENT.

CONTEXT-ELEMENT-OBJECTIVE

The focus, in terms of a CONTEXT-ELEMENT, of a specific TOPIC-MAP.

CONTEXT-OBJECTIVE-ELEMENT

The focus of a certain CONTEXT-ELEMENT, in terms of a REFERENCE-ELEMENT.

CONTEXT-RESOURCE-ELEMENT

A REFERENCE-ELEMENT that is required, requested, allocated or otherwise used in a specific CONTEXT-ELEMENT.

CONTROL-ELEMENT

A CONTROL-ELEMENT is introduced to adequately support the administration of REFERENCE-ELEMENTS.

CONTROL-REFERENCE-ELEMENT

An association between an ORGANISATION-ELEMENT and a certain REFERENCE-ELEMENT.

D

Begriff

Definition

DATA-REFERENCE-ELEMENT

A REFERENCE-ELEMENT which addresses the data elements administered by the data management.

DATETIME-CONTEXT-ELEMENT

An association between a certain DATETIME-ELEMENT and a CONTEXT-ELEMENT.

DATETIME-ELEMENT

The designation of a chronological point.

DISAGGREGATION-ELEMENT

The representation of the behavior of an aggregated unit in terms of its component entities. If the aggregate representation did not maintain state representations of the individual entities, then the decomposition into the entities can only be notional.

E

Begriff

Definition

EVALUATION-CRITERIA-DETAIL

An element of an EVALUATION-CRITERIA-LIST.

EVALUATION-CRITERIA-LIST

A list of selected evaluation criteria or requirements.

EVALUATION-ELEMENT

An EVALUATION-ELEMENT describes the evaluation (e.g. validation & verification) results for a certain REFERENCE-ELEMENT.

EVALUATION-REFERENCE-ELEMENT

An association between an EVALUATION-ELEMENT and a certain REFERENCE-ELEMENT.

EVENT-ELEMENT

A change of object attribute value, an interaction between objects, an instantiation of a new object, or a deletion of an existing object that is associated with a particular point on the federation time axis.

EXERCISE-ELEMENT

The execution of a simulation configured with specific parameters, characteristic data, initial conditions, players, and external systems intended to represent a specific or general scenario.

M

Begriff

Definition

METADATA-ELEMENT

A DATA-REFERENCE-ELEMENT, which denotes meta data referred as ISO IRDS Framework Layer 2 data. An example is an entity, attribute or other data element of a certain data model.

METADATA-ELEMENT-BUSINESS-RULE

A METADATA-ELEMENT-RULE which references a certain business rule for one or more METADATA-ELEMENTs.

METADATA-ELEMENT-BUSINESS-RULE-DETAIL

A detail of a certain METADATA-ELEMENT-BUSINESS-RULE expressed in terms of METADATA-ELEMENTs.

METADATA-ELEMENT-ESTABLISHMENT

An ESTABLISHMENT, which describes the assigned topology of METADATA-ELEMENTs.

METADATA-ELEMENT-RULE

The entity METADATA-ELEMENT-RULE denotes constraints applied to METADATA-ELEMENTs.

METADATA-ELEMENT-VALIDATION

A REFERENCE-ELEMENT-RULE which describes validation rules for a certain METADATA-ELEMENT.

O

Begriff

Definition

ORGANISATION-ELEMENT

An administrative or functional structures, that is responsible in a certain role for a specific REFERENCE-ELEMENT.

ORGANISATION-REFERENCE-ELEMENT

An association between an ORGANISATION-ELEMENT and a certain REFERENCE-ELEMENT.

P

Begriff

Definition

POST-ELEMENT

An ORGANISATION-ELEMENT with a set of duties that can be fulfilled by one person.

R

Begriff

Definition

REAL-DATETIME-ELEMENT

The designation of a chronological point in the real world.

REFERENCE-ELEMENT

The entity REFERENCE-ELEMENT represents the root entity to all reference data elements designed for data management purposes.

REFERENCE-ELEMENT-OBJECTIVE

The focus, in terms of a REFERENCE-ELEMENT of a specific TOPIC-MAP.

S

Begriff

Definition

SCENARIO-ELEMENT

An initial set of conditions and time line of significant events imposed on trainees or systems to achieve exercise objectives.

SIMULATION-PROCESS-ELEMENT

The imitative representation of the actions of platform(s), munitions(s), and life form(s) by computer program(s) in accordance with a mathematical model.

SYSTEM

A reference which denotes an IT-System as a whole.

SYSTEM-COMPONENT

A reference which denotes a certain component of an IT-System.

SYSTEM-METHOD

A reference which denotes algorithms, elementary software methods, services or procedures of an IT-System.

SYSTEM-MODULE

A reference which denotes a certain module of an IT-System.

SYSTEM-REFERENCE-ELEMENT

A REFERENCE-ELEMENT which denotes IT-Systems, and its parts.

SYSTEM-REFERENCE-ELEMENT-ESTABLISHMENT

An ESTABLISHMENT which describes the assigned topology of SYSTEM-REFERENCE-ELEMENTs.

SYSTEM-REFERENCE-ELEMENT-INTERFACE

An association between a certain SYSTEM-REFERENCE-ELEMENT and a specific DATA-REFERENCE-ELEMENT, used to describe the systems data interface.

T

Begriff

Definition

TOPIC-MAP

A topic map of a semantic network.

TOPIC-MAP-ASSOCIATION

A relationship of a TOPIC-MAP as a subject with another TOPIC-MAP as an object in order to specify a semantic network.

TOPIC-MAP-OBJECTIVE

The focus, in terms of REFERENCE-ELEMENTs or CONTEXT-ELEMENTs , of a specific TOPIC-MAP.

TOPIC-MAP-RESOURCE

A DATA-REFERENCE-ELEMENT that is required to describe a certain TOPIC-MAP in a standard manner (applicable to SDEs).

TOPIC-MAP-RESOURCE-CROSSWALK

The semantic crosswalk, from a specific DATA-REFERENCE-ELEMENT as a resource in a certain TOPIC-MAP to an objective of that TOPIC-MAP.

U

Begriff

Definition

UNIT-ELEMENT

An administrative or functional structure within a data management organisation, whose structure is prescribed by competent authority.

USER-DOMAIN-ELEMENT

The realm of knowledge describing a user's environment to which a model, simulation, or simulation federation will be applied.

USER-DOMAIN-REFERENCE-ELEMENT

An association between a USER-DOMAIN-ELEMENT and a REFERENCE-ELEMENT, to indicate the functional area or domain it is used in.

V

Begriff

Definition

VALIDATION-CONTEXT

The process of determining the degree to which a model and its associated data are an accurate representation of the real world from the perspective of the intended uses of the model.

VALIDATION-ELEMENT

Evaluation data for a certain REFERENCE-ELEMENT, which result from a process of determining the degree to which a model and its associated data are an accurate representation of the real world from the perspective of the intended use of the model.

VERIFICATION-CONTEXT

The process of determining that a model implementation and its associated data accurately represent the developer's conceptual description and specifications.

VERIFICATION-ELEMENT

Evaluation data for a certain REFERENCE-ELEMENT, which result from a process of determining that a model implementation and its associated data accurately represent the developer's conceptual description and specifications.

VIRTUAL-DATETIME-ELEMENT

The designation of a chronological point in a simulation-based activitity.
