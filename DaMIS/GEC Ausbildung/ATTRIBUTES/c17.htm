Attribut: capability-description-code

| Index | 


Definition
The specific value that represents or denotes the detailed class of a CAPABILITY.
Attributwerte
Bed count The numeric value denoting the number of hospital beds, fully outfitted with the necessary equipment and nursing personnel, available.
Breaching rate The numeric value which denotes the rate at which an object can be breached.
Breaching time The time normally required to execute a breaching.
Bulk liquid The numeric value representing the maximum amount of any liquid which can be transported.
Bulk volume The numeric value representing the maximum volume of any cargo which can be transported.
Bulk weight The numeric value representing the maximum weight of any cargo which can be transported.
Burst fire rate The numeric value which denotes the rate at which projectiles can be fired in a short but rapid burst.
Construction rate The numeric value which denotes the rate at which construction ob an object can be accomplished.
Construction time The time normally required to execute a construction.
Demolition rate The numeric value which denotes the rate at which demolition of an object can be accomplished.
Demolition time The time normally required to execute a demolition.
Low-planning-speed The average velocity that can be achieved, which is considered low for planning purposes, depending on the planned or given fuel consumption rate.
Lying persons count The numeric value representing the maximum number of prostrate persons (excluding the crew) which can be transported.
Maintenance station count The numeric value denoting the number of operational maintenance stations, fully outfitted with the necessary equipment and maintenance personnel, available for the purpose of repairing and servicing materiel.
Maximum altitude The highest elevation above sea-level at which a specified aircraft can fly.
Maximum cargo height The one-dimensional linear measurement that represents the extreme vertical distance, measured from the lowest to the highest reference point, of any object which can be transported.
Maximum cargo length The one-dimensional linear measurement that represents the extreme horizontal distance, measured from side to side and perpendicular to the central axis, of any object which can be transported.
Maximum cargo width The one-dimensional linear measurement that represents the extreme horizontal distance, measured from side to side and parallel to the central axis, of any object which can be transported.
Maximum depth The one-dimensional linear measurement that represents the greatest depth of any body of water.
Maximum fire rate The numeric value which denotes the highest rate at which projectiles can be fired for a limited period of time.
Maximum fording depth The one-dimensional linear measurement denoting the largest depth of any crossing through a flow or body of water (e.g., river, canal) which can be successfully traversed over the floor of that body.
Maximum obstacle gradient angle The numeric value representing the gradient of the steepest slope which can be climbed or descended head-on.
Maximum range The longest distance which can be achieved.
Maximum side slope angle The numeric value representing the gradient of the steepest slope which can be traversed in a direction perpendicular to that slope.
Maximum speed The highest velocity in any direction which can be achieved for a prolonged period of time.
Maximum trench width The one-dimensional linear measurement that represents the largest horizontal distance of any aperture which can be successfully navigated.
Maximum-cargo-weight The maximum weight that can be transported
Maximum-rate-of-climb The one-dimensional linear measurement that represents the greatest increase of altitude within a given period.
Maximum-steering-angle The maximum angle between the actual direction and the possible steering of any object.
Maximum-subsurface-depth The one-dimensional linear measurement that represents the greatest depth at which an object can move or operate properly below the surface of water.
Medium-planning-speed The average velocity that can be achieved, which is considered medium for planning purposes, depending on the planned or given fuel consumption rate.
Minimum depth The one-dimensional linear measurement that represents the smallest depth of any body of water.
Minimum landing distance The shortest stretch of unblocked and flat terrain or water required by an aircraft for it to be able to land.
Minimum range The shortest distance which can be achieved.
Minimum take off distance The shortest stretch of unblocked and flat terrain or water required by an aircraft for it to be able to take off.
Minimum-altitude The lowest elevation at which a specified aircraft can fly.
Minimum-subsurface-depth The one-dimensional linear measurement that represents the smallest depth at which an object can move or operate properly.
Operating table count The numeric value denoting the number of operational operating rooms, fully outfitted with the necessary equipment and surgical personnel, available to treat injuries and illness.
Pallet count The numeric value representing the maximum number of standard NATO pallets which can be transported.
Planning range The average distance that can be achieved which is considered normal for planning purposes.
Planning speed The average velocity that can be achieved which is considered normal for planning purposes.
Sitting persons count The numeric value representing the maximum number of sitting persons (excluding the crew) which can be transported.
Sustained fire rate The numeric value which denotes the rate at which projectiles can be fired repeatedly for a prolonged period of time.
